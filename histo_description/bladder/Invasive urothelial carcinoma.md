# Invasive urothelial carcinoma

Microscopic (histologic) description

* Neoplastic cells arranged in irregular nests or single cells invading the lamina propria and muscularis propria
* Retraction artifact is often seen and can mimic vascular invasion ([Am J Surg Pathol 2001;25:356](https://www.ncbi.nlm.nih.gov/pubmed/11224606))
* High grade nuclear features: nuclear pleomorphism, hyperchromasia, high N/C ratio with frequent mitotic figures ([Am J Surg Pathol 2001;25:356](https://www.ncbi.nlm.nih.gov/pubmed/11224606))
  + Note: [nested urothelial carcinoma](https://www.pathologyoutlines.com/topic/bladdernested.html) demonstrates bland, low grade cytology ([Hum Pathol 2019;94:11](https://www.ncbi.nlm.nih.gov/pubmed/31669177))
