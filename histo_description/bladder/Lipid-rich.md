# Lipid-rich

Microscopic (histologic) description

* Large signet ring lipoblast-like cells with single or multiple clear cytoplasmic vacuoles indenting the nucleus ([Eur Urol 2016;70:93](https://www.ncbi.nlm.nih.gov/pubmed/26935559))
* Lipid component comprises 10 - 50% of the tumor ([Am J Surg Pathol 2010;34:371](https://www.ncbi.nlm.nih.gov/pubmed/20139762))
