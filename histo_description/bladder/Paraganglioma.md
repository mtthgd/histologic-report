# Paraganglioma

Microscopic (histologic) description

* Typical morphology: epithelioid cells with abundant, eosinophilic / amphophilic, granular cytoplasm and round / ovoid, vesicular / salt and pepper nuclei, arranged in a nested / zellballen pattern and separated by fibrovascular septae with sustentacular cells ([Am J Surg Pathol 2004;28:94](https://www.ncbi.nlm.nih.gov/pubmed/14707870), [World J Clin Cases 2014;2:591](https://www.ncbi.nlm.nih.gov/pubmed/25325073), [Endocr Pathol 2022;33:90](https://www.ncbi.nlm.nih.gov/pubmed/35285002))
* May have focal pleomorphism, nuclear hyperchromasia, low mitotic activity or clear cytoplasm
