# Infantile myofibromatosis

Microscopic (histologic) description

* Same as [soft tissue myofibroma](https://www.pathologyoutlines.com/topic/softtissuemyopericytoma.html) - proliferation of spindle cells with pink cytoplasm, myxoid background
* Cells arranged in nodules with slit-like vascular spaces or hemangiopericytomatous pattern
