# Arsenic toxicity

Microscopic (histologic) description

* Peripheral smear: leukopenia including granulocytopenia and absolute eosinophilia, and profound anemia (normocytic or macrocytic) with coarse basophilic stippling and karyorrhexis ([N Engl J Med 1965;273:18](https://www.ncbi.nlm.nih.gov/pubmed/14297088), [West J Med 1983;139:219](https://www.ncbi.nlm.nih.gov/pubmed/6636735), [Am J Clin Pathol 1984;81:533](https://www.ncbi.nlm.nih.gov/pubmed/6702757), [Blut 1985;50:51](https://www.ncbi.nlm.nih.gov/pubmed/3967100))
* Dyserythropoiesis including megaloblastic features and typical karyorrhexis in erythropoietic cells, accompanied by basophilic stippling and impairment of mitoses in megakaryocytes and granulopoietic components
* Bone marrow biopsy shows nonspecific changes
