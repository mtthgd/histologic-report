# Copper deficiency

Microscopic (histologic) description

* Resembles myelodysplasia or sideroblastic anemia
* Presence of cytoplasmic vacuoles is suggestive ([Ann Clin Biochem 2005;42:227](https://www.ncbi.nlm.nih.gov/pubmed/15949160), [Am J Clin Pathol 1992;97:665](https://www.ncbi.nlm.nih.gov/pubmed/1575212))
