# Q fever

Microscopic (histologic) description

* Suggestive finding is a fibrin ring / doughnut granuloma defined as a small, nonnecrotizing granuloma with ring-like structure composed of fibrinoid material, often with a central fat vacuole ([Arch Pathol Lab Med 1986;110:1017](https://www.ncbi.nlm.nih.gov/pubmed/3778120))
* Rarely hemophagocytosis ([Clin Infect Dis 1995;21:1029](https://www.ncbi.nlm.nih.gov/pubmed/8645795))
