# AML with KAT6ACREBBP

Microscopic (histologic) description

* Morphologically, blasts often show myelomonocytic or monocytic differentiation ([Clin Case Rep 2014;2:333](https://www.ncbi.nlm.nih.gov/pubmed/25548641))
* Blasts are positive for myeloperoxidase (MPO) and alpha naphthyl butyrate esterase (ANB)
* Erythrophagocytosis by the leukemic blasts is seen in some cases ([Leukemia 2009;23:934](https://www.ncbi.nlm.nih.gov/pubmed/19194466), [Clin Case Rep 2014;2:333](https://www.ncbi.nlm.nih.gov/pubmed/25548641))
