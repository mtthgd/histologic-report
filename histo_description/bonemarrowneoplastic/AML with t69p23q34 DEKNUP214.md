# AML with t69p23q34 DEKNUP214

Microscopic (histologic) description

* Bone marrow core biopsy is often hypercellular with a diffuse infiltration of immature mononuclear cells
* While basophilia has been a classic finding with this entity, it is not actually observed as frequently and tends to be mild ([Mod Pathol 2021;34:1143](https://www.ncbi.nlm.nih.gov/pubmed/33558656), [Int J Lab Hematol 2017;39:409](https://www.ncbi.nlm.nih.gov/pubmed/28318095))
* Dysplasia as present in almost all cases ([Int J Lab Hematol 2017;39:409](https://www.ncbi.nlm.nih.gov/pubmed/28318095))
