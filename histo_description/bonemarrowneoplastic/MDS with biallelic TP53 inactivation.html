Microscopic (histologic) description
  • Blast count must be < 19% in bone marrow
  • Bone marrow cellularity is typically hyper / normocellular
  • Significant dysplasia in 1 or more of the myeloid lineages
    • Dysmegakaryopoiesis
    • Dyserythropoiesis
    • Dysgranulopoiesis
