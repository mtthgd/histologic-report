# Myeloid or lymphoid neoplasms with germline RUNX1 mutation

Microscopic (histologic) description

* Hypocellular or normocellular bone marrow for patient's age ([Pediatr Dev Pathol 2019;22:315](https://www.ncbi.nlm.nih.gov/pubmed/30600763), [Haematologica 2017;102:1661](https://www.ncbi.nlm.nih.gov/pubmed/28659335))
* Atypical but not dysplastic small megakaryocytes with hypolobated nuclei ([Pediatr Dev Pathol 2019;22:315](https://www.ncbi.nlm.nih.gov/pubmed/30600763), [Haematologica 2017;102:1661](https://www.ncbi.nlm.nih.gov/pubmed/28659335))
* Eosinophilia ([Haematologica 2017;102:1661](https://www.ncbi.nlm.nih.gov/pubmed/28659335))
