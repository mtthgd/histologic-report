# Supernumerary  accessory tissue

Microscopic (histologic) description

* Same histologic features as normal nipple, including hyperpigmentation, slight hyperkeratosis, pilosebaceous structure of Montgomery tubercles, smooth muscle, Toker cells and possibly breast lobules and ducts ([J Cutan Pathol 2003;30:256](https://www.ncbi.nlm.nih.gov/pubmed/12680957))
