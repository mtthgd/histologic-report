# Tuberculosis

Microscopic (histologic) description

* Epithelioid cell granulomas
* Langhans giant cells
* Caseous necrosis
* Lymphohistiocytic aggregates
* Fibrosis can be seen
* TB mastitis affects ducts more than lobules ([Turk Patoloji Derg 2022;38:40](https://www.ncbi.nlm.nih.gov/pubmed/34558655), [Indian J Surg 2015;77:1378](https://www.ncbi.nlm.nih.gov/pubmed/27011568))
