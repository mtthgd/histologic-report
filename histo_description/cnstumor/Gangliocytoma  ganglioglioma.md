# Gangliocytoma  ganglioglioma

Microscopic (histologic) description

* Ganglioglioma:
  + Biphasic tumor with variable mixture of mature appearing ganglion-like cells and atypical glial cells
  + Ganglion-like cells demonstrate dysmorphic features:
    - Binucleation
    - Neuron clustering (kissing neurons)
    - Cytomegaly with ballooning cytoplasm
    - Peripheral accumulation of Nissl substance
    - Lack of organized cytoarchitecture ([Acta Neuropathol 1994;88:166](https://www.ncbi.nlm.nih.gov/pubmed/7985497))
  + Atypical glial cells are hyperchromatic, moderately enlarged and resemble those of fibrillary astrocytomas, pilocytic astrocytoma or oligodendroglioma ([Hum Pathol 2016;49:107](https://www.ncbi.nlm.nih.gov/pubmed/26826417), [Clin Interv Aging 2014;9:1981](https://www.ncbi.nlm.nih.gov/pubmed/25429209))
  + Eosinophilic granular bodies more common than Rosenthal fibers ([Acta Neuropathol 1994;88:166](https://www.ncbi.nlm.nih.gov/pubmed/7985497))
  + Other common findings:
    - Dystrophic calcifications
    - Perivascular lymphoid infiltrates
    - Prominent capillary network
    - Low to absent mitotic activity ([J Neuropathol Exp Neurol 2002;61:575](https://www.ncbi.nlm.nih.gov/pubmed/12125736))
  + Compact growth but microscopic infiltration can be seen along the tumor edge ([Acta Neuropathol 2014;128:39](https://www.ncbi.nlm.nih.gov/pubmed/24858213), [Nat Rev Neurol 2016;12:732](https://www.ncbi.nlm.nih.gov/pubmed/27857123))
  + Rare cases with anaplastic features (anaplastic ganglioglioma) have been reported:
    - Tumors with ganglioglioma-like histology and increased mitoses, microvascular proliferation, necrosis
    - Many reported cases lack molecular analyses to exclude other high grade gliomas ([J Neuropathol Exp Neurol 2016;75:971](https://www.ncbi.nlm.nih.gov/pubmed/27539475), [Neuro Oncol 2017;19:678](https://www.ncbi.nlm.nih.gov/pubmed/28453747))
    - Anaplastic ganglioglioma is a term that should be used with caution, especially if complete molecular characterization is not available ([Neuropathol Appl Neurobiol 2022;48:e12847](https://www.ncbi.nlm.nih.gov/pubmed/35977725))
* Gangliocytoma:
  + Generally similar to that of ganglioglioma except lacking atypical neoplastic glial cells
  + Composed almost entirely of large, dysmorphic ganglion cells (as described above), often arranged in irregular clusters
