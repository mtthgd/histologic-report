# Infant type hemispheric glioma

Microscopic (histologic) description

* Tumor cells with astrocytic morphology that may include spindle cells and gemistocytes ([Cancer Discov 2020;10:942](https://www.ncbi.nlm.nih.gov/pubmed/32238360))
* In general, tumor cells do not infiltrate adjacent brain parenchyma ([Cancer Discov 2020;10:942](https://www.ncbi.nlm.nih.gov/pubmed/32238360))
* Some tumors may exhibit ependymal differentiation ([Cancer Discov 2020;10:942](https://www.ncbi.nlm.nih.gov/pubmed/32238360), [Neuro Oncol 2015;17:1365](https://www.ncbi.nlm.nih.gov/pubmed/25795305))
* Can have morphologic overlap with desmoplastic infantile ganglioglioma / astrocytoma ([Childs Nerv Syst 2023;39:1861](https://www.ncbi.nlm.nih.gov/pubmed/36707425))
* High grade features such as palisading necrosis, mitotic activity and microvascular proliferation ([Cancer Discov 2020;10:942](https://www.ncbi.nlm.nih.gov/pubmed/32238360))
* There can be low grade components or occasional ganglion cells ([Cancer Discov 2020;10:942](https://www.ncbi.nlm.nih.gov/pubmed/32238360), [Nat Commun 2019;10:4343](https://www.ncbi.nlm.nih.gov/pubmed/31554817), [Pediatr Blood Cancer 2020;67:e28015](https://www.ncbi.nlm.nih.gov/pubmed/31556208), [BMJ Case Rep 2019;12:e228248](https://www.ncbi.nlm.nih.gov/pubmed/30709888))
