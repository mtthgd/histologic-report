# Papillary craniopharyngioma

Microscopic (histologic) description

* Low power highlights papillary configuration with cauliflower-like morphology
* Solid sheets of well differentiated nonkeratinizing squamous epithelium ([J Neuropathol Exp Neurol 2017;76:126](https://www.ncbi.nlm.nih.gov/pubmed/28069929), [J Neurosurg 1995;83:206](https://www.ncbi.nlm.nih.gov/pubmed/7616262))
* Crude papillae around fibrovascular cores
* Small collagenous whorls
