# Subependymoma

Microscopic (histologic) description

* Nuclear clustering is typical; nuclei are bland, euchromatic, small, round to oval ([Neurol India 2012;60:379](https://www.ncbi.nlm.nih.gov/pubmed/22954972))
* Abundant densely fibrillar, paucicellular background ([Diagn Cytopathol 2018;46:258](https://www.ncbi.nlm.nih.gov/pubmed/29024543))
* Common
  + Microcysts containing mucoid material, especially near foramen of Monro ([Diagn Cytopathol 2018;46:258](https://www.ncbi.nlm.nih.gov/pubmed/29024543))
  + Calcifications, especially in posterior fossa subependymomas ([AJR Am J Roentgenol 1995;165:1245](https://www.ncbi.nlm.nih.gov/pubmed/7572512))
  + Sclerotic and ectatic blood vessels, hemorrhage and hemosiderin deposits ([Arch Pathol Lab Med 1999;123:306](https://www.ncbi.nlm.nih.gov/pubmed/10320142))
* Occasional
  + Focal perivascular pseudorosettes ([J Neurooncol 2007;85:297](https://www.ncbi.nlm.nih.gov/pubmed/17569000))
  + Nuclear pleomorphism ([Arch Pathol Lab Med 1999;123:306](https://www.ncbi.nlm.nih.gov/pubmed/10320142))
  + Proliferative microvascular abnormalities
* Rare
  + Low level mitotic activity
  + Necrosis ([J Neurooncol 2007;85:297](https://www.ncbi.nlm.nih.gov/pubmed/17569000))
* Exceptionally rare
  + Melanotic pigmentation ([Am J Surg Pathol 1990;14:729](https://www.ncbi.nlm.nih.gov/pubmed/2378394))
  + Sarcomatous change ([Neurosurgery 1991;28:761](https://www.ncbi.nlm.nih.gov/pubmed/1876259), [Am J Surg Pathol 2008;32:699](https://www.ncbi.nlm.nih.gov/pubmed/18347506))
* Mixed histologic pattern (subependymoma together with another glial tumor)
  + Mixed ependymoma subependymoma, the most common (13.7%) ([J Neurosurg 2021;136:736](https://www.ncbi.nlm.nih.gov/pubmed/34416731))
  + Subependymoma with elements of fibrillary astroglial or (rarely) gemistocytic morphology ([J Neurooncol 2007;85:297](https://www.ncbi.nlm.nih.gov/pubmed/17569000))
