# Adenoma-like adenocarcinoma

Microscopic (histologic) description

* Invasive carcinoma with architectural and cytologic features resembling villous adenoma
* May have traditional invasive component or invade by pushing
* Mucinous features often present
* Epithelial islands in desmoplastic stroma is a helpful finding ([Am J Surg Pathol 2004;28:1460](https://www.ncbi.nlm.nih.gov/pubmed/15489649))
* May also have a component of conventional adenocarcinoma ([Hum Pathol 2021;107:9](https://www.ncbi.nlm.nih.gov/pubmed/32991929))
