# Brainerd diarrhea

Microscopic (histologic) description

* Surface epithelial lymphocytosis (similar to lymphocytic colitis) with no architectural distortion, no degenerative changes, no thickened subepithelial collagen ([Am J Surg Pathol 1996;20:1102](https://www.ncbi.nlm.nih.gov/pubmed/8764747))
