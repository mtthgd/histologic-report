# Mycobacterium avium intracellulare

Microscopic (histologic) description

* Histological patterns of colonic MAC are similar to duodenal / small intestine MAC infection; the findings can be classified as typical or atypical patterns ([Ann Diagn Pathol 2020;49:151638](https://www.ncbi.nlm.nih.gov/pubmed/33069083))
* Typical pattern
  + Diffuse lamina propria expansion by foamy histiocytes
* Atypical patterns
  + Scattered single histiocytes
  + Loosely organized epithelioid granuloma-like collections
  + Deep submucosal involvement with mucosal sparing ([Case Rep Gastrointest Med 2019;2019:8129597](https://www.ncbi.nlm.nih.gov/pubmed/31885949), [Diseases 2018;6:36](https://www.ncbi.nlm.nih.gov/pubmed/29738438))
* Histiocytes may range from large round, polygonal cells with distended eosinophilic granular / foamy cytoplasm (typical) to oval spindly cells with pale vacuolated cytoplasm
* Intracytoplasmic bacilli are usually numerous, densely packed and better highlighted on AFB or PAS special stains
