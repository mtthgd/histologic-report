# Tactile corpuscle-like bodies

Microscopic (histologic) description

* Unencapsulated clusters or nodules of bland spindle cells within the lamina propria ([Int J Surg Pathol 2012;20:79](https://www.ncbi.nlm.nih.gov/pubmed/21632642))
* Characteristic lamellated or fibrillary eosinophilic cytoplasm ([Int J Surg Pathol 2012;20:79](https://www.ncbi.nlm.nih.gov/pubmed/21632642))
* Nuclei are peripherally oriented around the nodules ([Int J Surg Pathol 2012;20:79](https://www.ncbi.nlm.nih.gov/pubmed/21632642))
