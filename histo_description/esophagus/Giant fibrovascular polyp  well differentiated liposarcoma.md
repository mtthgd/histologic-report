# Giant fibrovascular polyp  well differentiated liposarcoma

Microscopic (histologic) description

* Fibroadipose tissue with slightly enlarged, hyperchromatic stromal cells, covered with intact squamous mucosa ([Mod Pathol 2018;31:337](https://www.ncbi.nlm.nih.gov/pubmed/28984298))
* Mostly well differentiated type / atypical lipomatous tumor ([Mod Pathol 2018;31:337](https://www.ncbi.nlm.nih.gov/pubmed/28984298))
* Occasionally dedifferentiated type ([Mod Pathol 2018;31:337](https://www.ncbi.nlm.nih.gov/pubmed/28984298))
