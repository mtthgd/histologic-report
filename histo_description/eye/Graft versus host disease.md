# Graft versus host disease

Microscopic (histologic) description

* Reduced goblet cells
  ([Eur J Ophthalmol 2006;16:17](https://www.ncbi.nlm.nih.gov/pubmed/16496240))
* Pseudomembrane due to loss of conjunctival epithelium
* May have keratinization
* Increased ICAM-1 expression
  ([Eur J Ophthalmol 2006;16:17](https://www.ncbi.nlm.nih.gov/pubmed/16496240))
* Reduced mucosal microvilli
  ([Bone Marrow Transplant 2012;47:416](https://www.ncbi.nlm.nih.gov/pubmed/21572465))
