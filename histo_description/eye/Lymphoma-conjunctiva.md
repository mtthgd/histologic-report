# Lymphoma-conjunctiva

Microscopic (histologic) description

* MALT: well differentiated, monoclonal small B lymphocytes, occasionally exhibits overt monocytoid cytology, prominent plasmacytic features or lymphoepithelial lesions ([Am J Surg Pathol 2007;31:792](https://www.ncbi.nlm.nih.gov/pubmed/17460465))
