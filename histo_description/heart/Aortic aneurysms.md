# Aortic aneurysms

Microscopic (histologic) description

* Medial degeneration ([Histopathology 1990;16:557](https://www.ncbi.nlm.nih.gov/pubmed/2376398))
  + Loss / disorganization of elastic lamellae
  + Loss of smooth muscle cells
  + Mucoid extracellular matrix accumulation (MEMA)
  + Medial fibrosis
* Border between tunica media and tunica intima may be obscured
* Inflammatory reaction ([JVS Vasc Sci 2021;2:260](https://www.ncbi.nlm.nih.gov/pubmed/34825232))
  + Lymphocyte and macrophage infiltration
  + Medial neovascularization
  + Increased proteolysis
* Increasing proteoglycan deposition
* Atherosclerotic lesions ([Cardiovasc Pathol 2015;24:267](https://www.ncbi.nlm.nih.gov/pubmed/26051917))
  + Lipid deposits, foam cells, cholesterol clefts, eosinophilic debris, calcifications or neovascularization
* Increase of various matrix metalloproteinases (MMPs) and cadherin ([Histopathology 1990;16:557](https://www.ncbi.nlm.nih.gov/pubmed/2376398))
* Increased collagenase / elastase activity
* There may be luminal fibrin thrombus present ([Cardiovasc Pathol 2015;24:267](https://www.ncbi.nlm.nih.gov/pubmed/26051917))
