# Hypertrophic cardiomyopathy

Microscopic (histologic) description

* Hallmarks of histological diagnosis include myocyte hypertrophy, fibrosis and disarray ([Int J Mol Sci 2024;25:1275](https://www.ncbi.nlm.nih.gov/pubmed/38279275))
* Cells are abnormally branched with hyperchromatic and enlarged nuclei ([Int J Mol Sci 2024;25:1275](https://www.ncbi.nlm.nih.gov/pubmed/38279275))
* Fibrosis is present interstitially between myocytes, around capillaries in a perivascular pattern or forms scar / replacement tissue after myocyte death ([Int J Mol Sci 2024;25:1275](https://www.ncbi.nlm.nih.gov/pubmed/38279275))
