# Autoimmune lymphoproliferative syndrome

Microscopic (histologic) description

* Paracortical hyperplasia, expanded interfollicular areas; polyclonal plasmacytosis ([Am J Pathol 1998;153:1541](https://www.ncbi.nlm.nih.gov/pubmed/9811346))
* Frequent mitotic figures
* Often florid follicular hyperplasia with focal progressive transformation of germinal centers
* Occasional follicular involution
