# Paroxysmal nocturnal hemoglobinuria PNH

Microscopic (histologic) description

* Bone marrow:
  + Bone marrow analysis is primarily done to determine if PNH arose in association with aplastic anemia (AA) or myelodysplastic syndrome (MDS) ([Pract Lab Med 2020;20:e00158](https://www.ncbi.nlm.nih.gov/pubmed/32195308), [Br J Haematol 2018;182:758](https://www.ncbi.nlm.nih.gov/pubmed/29974931))
  + There are no specific morphologic features for PNH ([Pract Lab Med 2020;20:e00158](https://www.ncbi.nlm.nih.gov/pubmed/32195308), [Br J Haematol 2018;182:758](https://www.ncbi.nlm.nih.gov/pubmed/29974931), [J Hematopathol 2013;6:71](https://doi.org/10.1007/s12308-013-0179-7))
  + Bone marrow cellularity may be increased, decreased or within a normal range for age ([J Hematopathol 2013;6:71](https://doi.org/10.1007/s12308-013-0179-7))
  + Erythroid hyperplasia, decreased myelopoeisis and decreased megakaryocytes are features that can be seen in PNH bone marrow ([J Hematopathol 2013;6:71](https://doi.org/10.1007/s12308-013-0179-7), [Pract Lab Med 2020;20:e00158](https://www.ncbi.nlm.nih.gov/pubmed/32195308), [Br J Haematol 2018;182:758](https://www.ncbi.nlm.nih.gov/pubmed/29974931), [Semin Hematol 2019;56:65](https://www.ncbi.nlm.nih.gov/pubmed/30573047))
