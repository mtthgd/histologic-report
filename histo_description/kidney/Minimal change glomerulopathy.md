# Minimal change glomerulopathy

Microscopic (histologic) description

* Glomerulus normal by light microscopy except for variable podocyte hypertrophy
* Protein reabsorption droplets in tubules ([Zhou: Silva's Diagnostic Renal Pathology, 2nd Edition, 2017](https://www.amazon.com/exec/obidos/ASIN/1316613984/pathologyoutl-20))
