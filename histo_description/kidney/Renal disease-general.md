* Renal allograft ([Transplantation 2018;102:1795](https://www.ncbi.nlm.nih.gov/pubmed/30028786), [Am J Transplant 2020;20:2318](https://www.ncbi.nlm.nih.gov/pubmed/32463180))
  + 2019 Banff classification for antibody mediated rejection (ABMR), borderline changes, T cell mediated rejection (TCMR) and polyomavirus nephropathy ([Am J Transplant 2020;20:2318](https://www.ncbi.nlm.nih.gov/pubmed/32463180))
    - Category 1: normal biopsy or nonspecific changes
    - Category 2: antibody mediated changes
      * Active ABMR
      * Chronic active ABMR; all 3 criteria must be met for diagnosis
      * Chronic (inactive) ABMR
    - Category 3: borderline (suspicious) for acute TCMR
    - Category 4: TCMR
      * Acute TCMR
      * Chronic active TCMR
    - Category 5: polyomavirus nephropathy
  + Infectious
    - Cytomegalovirus infection ([Am J Kidney Dis 2016;68:e35](https://www.ncbi.nlm.nih.gov/pubmed/27884284))
    - Polyomavirus nephropathy ([Am J Kidney Dis 2016;68:e37](https://www.ncbi.nlm.nih.gov/pubmed/27884285))
  + Calcineurin inhibitor nephrotoxicity ([Am J Kidney Dis 2017;69:e21](https://www.ncbi.nlm.nih.gov/pubmed/28434526))
