# Anaplastic sarcoma of the kidney

Microscopic (histologic) description

* Typical features
  + Undifferentiated cells
  + Spindle cell areas
  + Marked diffuse anaplasia
  + Benign or malignant cartilage or chondroid differentiation
  + Rhabdomyoblastic differentiation (desmin and myogenin positivity)
  + Cystic nephroma-like cystic component
* May have
  + Osteoid differentiation
  + Ganglioneuroblastic differentiation ([Pediatr Dev Pathol 2022;25:186](https://www.ncbi.nlm.nih.gov/pubmed/34515577))
* Staging criteria are the same as [nephroblastoma](https://www.pathologyoutlines.com/topic/kidneytumorwilmkids.html) ([Histopathology 2022;80:1026](https://www.ncbi.nlm.nih.gov/pubmed/35275409))
