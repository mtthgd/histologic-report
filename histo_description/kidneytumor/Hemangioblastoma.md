# Hemangioblastoma

Microscopic (histologic) description

* Sheets, lobulated or nested growth pattern
* Round to oval shaped cells containing microvacuoles in eosinophilic or clear cytoplasm (so called stromal cells), mimicking lipoblasts
* Complex capillary network comprises mainly thin walled vessels
* Some areas show spindle cells surrounding vessels (so called pericytomatous pattern) ([Int J Surg Pathol 2012;20:519](https://www.ncbi.nlm.nih.gov/pubmed/22271882))
* Focal areas may contain predominant spindle cells with minimal cytoplasm and vague vacuolization ([Am J Surg Pathol 2014;38:119](https://www.ncbi.nlm.nih.gov/pubmed/24145646))
* Usually mild pleomorphism; however, occasional marked nuclear pleomorphism, rhabdoid features or pseudointranuclear inclusions can be seen ([Am J Surg Pathol 2014;38:119](https://www.ncbi.nlm.nih.gov/pubmed/24145646), [Diagn Pathol 2012;7:39](https://www.ncbi.nlm.nih.gov/pubmed/22497861))
