# Foregut cyst

Microscopic (histologic) description

* The only ciliated cyst of the liver
* 4 layers can be appreciated
  + Ciliated pseudostratified columnar epithelium
  + Subepithelial connective tissue
  + Layer of smooth muscle
  + Fibrous capsule
* Pseudostratified lining of ciliated columnar epithelium with goblet cells
* Subepithelial loose connective tissue, fibrous and paucicellular ([Hum Pathol 2000;31:241](https://www.ncbi.nlm.nih.gov/pubmed/10685641))
* Smooth muscle layer composed of 1 - 3 discontinuous smooth muscle layers, surrounded by an outer fibrous capsule ([Hum Pathol 2000;31:241](https://www.ncbi.nlm.nih.gov/pubmed/10685641))
* Fibrous capsule with arterioles and nerve trunks can be seen
* Cyst wall can be disrupted with inflammatory changes in the surrounding liver parenchyma ([Hum Pathol 2000;31:241](https://www.ncbi.nlm.nih.gov/pubmed/10685641))
* Squamous metaplasia without evidence of dysplasia might occur ([Diagn Pathol 2015;10:81](https://www.ncbi.nlm.nih.gov/pubmed/26122082))
* Gastric antral metaplasia might be present ([Diagn Pathol 2015;10:81](https://www.ncbi.nlm.nih.gov/pubmed/26122082))
* Rarely, malignant transformation can occur and is associated with a bigger size ([Pathol Res Pract 2009;205:498](https://www.ncbi.nlm.nih.gov/pubmed/19410383), [Pathol Res Pract 2002;198:697](https://www.ncbi.nlm.nih.gov/pubmed/12498226), [Am J Surg Pathol 1984;8:467](https://www.ncbi.nlm.nih.gov/pubmed/6731667))
