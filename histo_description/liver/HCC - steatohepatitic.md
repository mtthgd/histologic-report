# HCC - steatohepatitic

Microscopic (histologic) description

* Hepatocellular carcinoma with ≥ 50% of tumor showing 3 or more of the following steatohepatitis-like features ([Am J Surg Pathol 2010;34:1630](https://www.ncbi.nlm.nih.gov/pubmed/20975341)):
  + Steatosis
  + Hepatocyte ballooning
  + Mallory-Denk bodies
  + Inflammation
  + Pericellular fibrosis
