# Intrahepatic cholangiocarcinoma small and large duct types

Microscopic (histologic) description

* iCCA consists of infiltrating well formed or cribriform glands in an abundant fibrous stroma ([Liver Int 2019;39:7](https://www.ncbi.nlm.nih.gov/pubmed/30882996), [Surg Pathol Clin 2018;11:403](https://www.ncbi.nlm.nih.gov/pubmed/29751883))
* Malignant glands are lined by cells with varying degrees of atypia and pleomorphism ([Liver Int 2019;39:7](https://www.ncbi.nlm.nih.gov/pubmed/30882996))
* Usually well differentiated adenocarcinoma with mild atypia, intracytoplasmic lumina and intraluminal cellular debris; however, focal atypia with marked pleomorphism can also be present ([Liver Int 2019;39:7](https://www.ncbi.nlm.nih.gov/pubmed/30882996))
* Commonly infiltrates between hepatic parenchymal cords at the periphery of the tumor
* Multicentricity and perineural invasion are common
* Histologic variants include mucinous, signet ring cell, clear cell, [lymphoepithelioma-like](https://www.pathologyoutlines.com/topic/livertumorlelhcc.html), thyroid follicular-like, adenosquamous and sarcomatoid
* Small duct type iCCA ([Liver Int 2019;39:7](https://www.ncbi.nlm.nih.gov/pubmed/30882996))
  + Peripheral hepatic parenchyma
  + Mass forming
  + Cuboidal cells forming small tubular or anastomosing glands
  + Variable lymphovascular / perineural invasion
  + CD56, N-cadherin, CRP positive
  + Subtypes include cholangiolocarcinoma, iCCA with ductal plate malformation-like pattern
* Large duct type iCCA ([Liver Int 2019;39:7](https://www.ncbi.nlm.nih.gov/pubmed/30882996))
  + Proximal to liver hilum
  + Periductal infiltrating or mass forming
  + Tall columnar cells that form large glands within open lumina; usually contain mucin
  + BilIN, IPNB, ITPN are precursors
  + Positive lymphovascular / perineural invasion
  + MUC5AC, MUC6, S100P and TFF1 positive
