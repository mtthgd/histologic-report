# Navajo hepatopathy

Microscopic (histologic) description

* The following features are characteristic but not pathognomonic:
  + Mixed macrovesicular and microvesicular steatosis with hepatocanalicular cholestasis ([J Pediatr 1999;135:482](https://www.ncbi.nlm.nih.gov/pubmed/10518083))
  + Ballooning hepatocellular degeneration indistinguishable from typical steatohepatitis
  + Red granular hepatocytes containing abnormal megamitochondria ([Acta Gastroenterol Belg 2016;79:463](https://www.ncbi.nlm.nih.gov/pubmed/28209105))
  + Scattered acidophil bodies ([Acta Gastroenterol Belg 2016;79:463](https://www.ncbi.nlm.nih.gov/pubmed/28209105))
  + Bridging fibrosis and cirrhosis with nonspecific portal / septal ductular reaction ([J Pediatr 1999;135:482](https://www.ncbi.nlm.nih.gov/pubmed/10518083))
  + Juxtaposed cirrhotic nodules appear fatty and oncocytic due to variable accumulations of fat and megamitochondria
