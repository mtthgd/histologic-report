# Secondary biliary cirrhosis

Microscopic (histologic) description

* Same as biliary pattern cirrhosis of other causes, with jigsaw architecture, ductular reaction and cholestasis ([Burt: Macsween's Pathology of the Liver, 7th Edition, 2018](https://www.amazon.com/exec/obidos/ASIN/0702066974/pathologyoutl-20)
* May show septal edema, feathery degeneration, bile infarcts, bile lakes
* Should not see florid duct lesions of primary biliary cholangitis
