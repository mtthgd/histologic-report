# Langerhans cell histiocytosis

Microscopic (histologic) description

* Macrophages, eosinophils, fibrosis and granulomas are found on H&E
* Number of eosinophils varies
* Langerhans cells are large and have a pale, somewhat eosinophilic cytoplasm and a convoluted nucleus that has a longitudinal crease and looks like a coffee bean ([Front Med (Lausanne) 2021;7:582581](https://www.ncbi.nlm.nih.gov/pubmed/33763431))
* Older lesions become fibrotic
* Can present as a stellate shaped scar
