# Large cell neuroendocrine carcinoma

Microscopic (histologic) description

* Neuroendocrine organoid architecture may include nesting with peripheral palisading, anastomosing trabeculae and rosette-like structures ([Mod Pathol 2022;35:36](https://www.ncbi.nlm.nih.gov/pubmed/34663914))
* > 10 mitoses/2 mm2, extensive / geographic necrosis
* Large cells (~3x size of small cell carcinoma) with abundant amphophilic cytoplasm, intercellular membranes, nuclear pleomorphism, variably coarse, granular or vesicular chromatin with prominent nucleoli ([Transl Lung Cancer Res 2020;9:860](https://www.ncbi.nlm.nih.gov/pubmed/32676352), [Mod Pathol 2022;35:36](https://www.ncbi.nlm.nih.gov/pubmed/34663914))
* Occasional rhabdoid features ([Pathol Int 2019;69:481](https://www.ncbi.nlm.nih.gov/pubmed/31237049),
  [J Cancer Res Ther 2015;11:657](https://www.ncbi.nlm.nih.gov/pubmed/26458655))
* Larger tumor cells than atypical carcinoid, high nuclear grade, increased mitotic activity and necrosis ([Arch Pathol Lab Med 2010;134:1628](https://www.ncbi.nlm.nih.gov/pubmed/21043816))
