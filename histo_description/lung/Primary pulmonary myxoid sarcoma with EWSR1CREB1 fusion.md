# Primary pulmonary myxoid sarcoma with EWSR1CREB1 fusion

Microscopic (histologic) description

* Lobulated architecture, reticular or lace-like pattern with anastomosing cords and strands, abundant myxoid stroma, often lightly basophilic ([Am J Surg Pathol 2011;35:1722](https://www.ncbi.nlm.nih.gov/pubmed/21997693))
* Oval, spindle to polygonal cells with minimal to moderate atypia, infrequent mitotic figures with the majority < 5/10 high power fields
* Admixed lymphoplasmacytic infiltrate
* Some cells are chondrocyte-like or mimicking physaliferous cells in chordoma ([Diagn Pathol 2020;15:15](https://www.ncbi.nlm.nih.gov/pubmed/32039736))
