# Rheumatoid lung disease

Microscopic (histologic) description

* Characteristic features of pulmonary rheumatoid nodules ([Biomed Res Int 2017;2017:7915340](https://www.ncbi.nlm.nih.gov/pubmed/29238722), [Clin Rheumatol 2019;38:3041](https://www.ncbi.nlm.nih.gov/pubmed/31273637), [Eur Radiol 2019;29:1684](https://www.ncbi.nlm.nih.gov/pubmed/30288558))
  + Palisading histiocytes
  + Cavitation
  + Multiplicity
  + Eosinophilic and basophilic necrosis
  + Subpleural location
* Characteristic features of the rim of rheumatoid nodules ([Clin Rheumatol 2019;38:3041](https://www.ncbi.nlm.nih.gov/pubmed/31273637))
  + Chronic inflammatory infiltrate
  + Fibrosis
* Characteristic features of lung parenchyma surrounding nodules ([Biomed Res Int 2017;2017:7915340](https://www.ncbi.nlm.nih.gov/pubmed/29238722), [Semin Respir Crit Care Med 2014;35:201](https://www.ncbi.nlm.nih.gov/pubmed/24668535))
  + Follicular bronchiolitis
  + Lymphoid aggregates
* Pleural disease ([Rheum Dis Clin North Am 2015;41:225](https://www.ncbi.nlm.nih.gov/pubmed/25836639))
  + Chronic pleuritis with fibrous thickening, lymphoplasmacytic infiltrates and organizing exudates
* Usual interstitial pneumonia ([J Clin Pathol 2013;66:896](https://www.ncbi.nlm.nih.gov/pubmed/23703852), [Chest 2009;136:1397](https://www.ncbi.nlm.nih.gov/pubmed/19892679), [J Clin Med 2019;8:2038](https://www.ncbi.nlm.nih.gov/pubmed/31766446), [Rheum Dis Clin North Am 2015;41:225](https://www.ncbi.nlm.nih.gov/pubmed/25836639))
  + Patchy interstitial fibrosis with architectural distortion and honeycombing
  + Fibroblast foci and temporal heterogeneity
  + Subpleural and basal predominance
* Nonspecific interstitial pneumonia (NSIP) ([Chest 2009;136:1397](https://www.ncbi.nlm.nih.gov/pubmed/19892679), [J Clin Med 2019;8:2038](https://www.ncbi.nlm.nih.gov/pubmed/31766446))
  + Relative preservation of lung architecture without honeycombing
    - Fibrotic NSIP: uniform interstitial thickening with dense or loose fibrosis
    - Cellular NSIP: with chronic inflammatory cell infiltrates
    - Mixed with both fibrosis and chronic interstitial inflammation
* Organizing pneumonia (OP) ([J Clin Med 2019;8:2038](https://www.ncbi.nlm.nih.gov/pubmed/31766446), [Reumatologia 2017;55:314](https://www.ncbi.nlm.nih.gov/pubmed/29491540))
  + Intra-alveolar plugs of organizing connective tissue (Masson bodies)
  + Preserved underlying lung architecture
  + Mild interstitial chronic inflammation
