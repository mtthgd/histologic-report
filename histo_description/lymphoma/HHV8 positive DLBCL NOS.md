# HHV8 positive DLBCL NOS

Microscopic (histologic) description

* Effacement of lymph node / splenic architecture by confluent sheets of large plasmablastic cells
* Large plasmablastic cells have amphophilic cytoplasm and vesicular, often eccentrically placed nuclei containing 1 or 2 prominent nucleoli ([Am J Clin Pathol 2017;147:171](https://www.ncbi.nlm.nih.gov/pubmed/28395104))
* High mitotic activity
