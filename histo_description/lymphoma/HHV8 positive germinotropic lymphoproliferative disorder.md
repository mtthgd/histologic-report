# HHV8 positive germinotropic lymphoproliferative disorder

Microscopic (histologic) description

* Small aggregates of or single plasmablasts / immunoblasts confined to the germinal center ([Ann Hematol 2020;99:2243](https://www.ncbi.nlm.nih.gov/pubmed/32307569))
* These are large atypical cells with anaplastic features ([Int J Surg Pathol 2020;28:804](https://www.ncbi.nlm.nih.gov/pubmed/32423260))
* Coinfected by HHV8 and EBV
* Atrophic and hyalinized follicles with vascular proliferation resembling [Castleman disease](https://www.pathologyoutlines.com/topic/lymphnodescastleman.html)
