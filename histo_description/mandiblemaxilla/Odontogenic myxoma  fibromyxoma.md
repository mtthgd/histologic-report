# Odontogenic myxoma  fibromyxoma

Microscopic (histologic) description

* Stellate, spindle shaped, round cells arranged haphazardly in abundant, fibrillary myxoid / mucoid stroma ([Head Neck Pathol 2020;14:1021](https://www.ncbi.nlm.nih.gov/pubmed/32506377))
* Infiltrates surrounding bone ([Eur J Clin Invest 2020;50:e13214](https://www.ncbi.nlm.nih.gov/pubmed/32048275))
* Small islands of inactive odontogenic epithelial rests can be seen ([Head Neck Pathol 2020;14:1021](https://www.ncbi.nlm.nih.gov/pubmed/32506377))
* If abundant collagen is present, the term fibromyxoma is used ([Head Neck Pathol 2020;14:1021](https://www.ncbi.nlm.nih.gov/pubmed/32506377), [Eur J Clin Invest 2020;50:e13214](https://www.ncbi.nlm.nih.gov/pubmed/32048275))
