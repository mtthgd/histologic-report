# True thymic hyperplasia

Microscopic (histologic) description

* Expansion of the normal component of the thymic gland ([Histopathology 2009;54:69](https://www.ncbi.nlm.nih.gov/pubmed/19054157))
* Normal cortex, medulla and Hassall corpuscles
* No neoplasm
* No increased reactive germinal centers ([Histopathology 2009;54:69](https://www.ncbi.nlm.nih.gov/pubmed/19054157))
* Lobular architecture and corticomedullary junction are preserved
