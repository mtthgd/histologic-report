# Inflammatory fibrous hyperplasia

Microscopic (histologic) description

* Hyperplastic keratinized squamous mucosa overlying dense nodular connective tissue and variable inflammatory infiltrate ([Acta Histochem 2016;118:451](https://www.ncbi.nlm.nih.gov/pubmed/27132705), [Head Neck Pathol 2019;13:103](https://www.ncbi.nlm.nih.gov/pubmed/30693455))
* Underlying connective tissue is variably collagenized with scant bland spindled fibroblasts ([Head Neck Pathol 2019;13:103](https://www.ncbi.nlm.nih.gov/pubmed/30693455))
* Pseudoepitheliomatous hyperplasia or secondary candidal colonization may be present
