# Torsion

Microscopic (histologic) description

* Early features
  + Hemorrhage and vascular congestion / thrombi
* Developed features
  + Hemorrhagic infarction with edema, acute hemorrhage and inflammatory cells
  + Ovarian parenchyma may show ghost cells
* Associated ovarian cyst / neoplasms
  + Teratoma
  + Cystadenomas
  + Functional cysts
  + Endometrioma
  + Fibroma / thecoma
  + Malignancy
* Recommendation: thorough sampling to exclude malignancy
* Reference: Nucci, Marisa R. Gynecologic Pathology: Foundations in Diagnostic Pathology, 2nd edition, 2017, pp. 545-575, Nucci, Marisa R. Diagnostic Pathology: Gynecological, 2nd edition.
