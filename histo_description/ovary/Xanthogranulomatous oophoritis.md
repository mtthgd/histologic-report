# Xanthogranulomatous oophoritis

Microscopic (histologic) description

* Usually shows characteristic dense fibrosis with infiltration of the ovarian stroma by sheets and nodules of foamy macrophages, many histiocytes, lymphocytes and neutrophils ([Iran J Pathol 2018;13:372](https://www.ncbi.nlm.nih.gov/pubmed/30636962))
