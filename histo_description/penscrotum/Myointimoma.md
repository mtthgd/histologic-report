# Myointimoma

Microscopic (histologic) description

* Multinodular or plexiform pattern; composed of occlusive intravascular myointimal proliferation ([Am J Surg Pathol 2000;24:1524](https://www.ncbi.nlm.nih.gov/pubmed/11075854))
* Nodules contain spindle or stellate shaped cells embedded in abundant fibromyxoid matrix or sometimes chondroid matrix ([Am J Surg Pathol 2007;31:1622](https://www.ncbi.nlm.nih.gov/pubmed/17895766))
* Cells have long eosinophilic cytoplasmic processes, blunt ended nuclei, fine chromatin and juxtanuclear vacuoles
* Foci of degenerative changes appear as ghost cells ([Am J Surg Pathol 2000;24:1524](https://www.ncbi.nlm.nih.gov/pubmed/11075854))
* No cytologic atypia, nuclear pleomorphism, prominent nucleoli or mitoses
* Residual smooth muscle bundles surrounding the tumor ([Actas Dermosifiliogr 2009;100:511](https://www.ncbi.nlm.nih.gov/pubmed/19709559))
* Overlying skin may show slight hyperkeratosis ([J Cutan Pathol 2009;36:817](https://www.ncbi.nlm.nih.gov/pubmed/19519617))
* No necrosis or significant inflammation
