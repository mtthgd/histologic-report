# Mesothelioma in situ pleura

Microscopic (histologic) description

* Layer of flat or cuboidal cells
* No cytological atypia or if present it should be minimal
* No invasion into the surrounding tissue
* Bland nuclear features, with some cells displaying a prominent nucleolus
* Usually without mitotic activity
* Small papillary projections are sometimes present
* Reference: [Mod Pathol 2020;33:297](https://www.ncbi.nlm.nih.gov/pubmed/31375770)
