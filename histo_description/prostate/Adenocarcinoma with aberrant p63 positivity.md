# Adenocarcinoma with aberrant p63 positivity

Microscopic (histologic) description

* The first series in 2008 by Osunkoya et al. described 21 cases on needle biopsy, including 8 with matched prostatectomy in which > 70% of cancer cells were p63 positive ([Am J Surg Pathol 2008;32:461](https://www.ncbi.nlm.nih.gov/pubmed/18300803))
  + Distinctive, atrophic-like morphology was noted
* 21 matched biopsy / prostatectomy cases were characterized ([Am J Surg Pathol 2013;37:1401](https://www.ncbi.nlm.nih.gov/pubmed/23774168))
  + Besides atrophic-like change, salient features that have been described include a basaloid appearance and high N:C ratio, with multilayered and sometimes spindled nuclei
  + Notably, at prostatectomy, 86% of cases had coexisting usual type p63 negative adenocarcinoma while 14% were pure
