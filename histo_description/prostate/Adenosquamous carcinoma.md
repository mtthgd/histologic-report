# Adenosquamous carcinoma

Microscopic (histologic) description

* In addition to the usual glandular prostate cancer component, there is a component of cells with eosinophilic cytoplasm
* Keratin pearls and intercellular bridges are required to diagnose squamous cell carcinoma
* Additional sarcomatoid transformation has been rarely reported ([Histopathology 2020;77:890](https://www.ncbi.nlm.nih.gov/pubmed/32639612), [Scientific World Journal 2006;6:2491](https://www.ncbi.nlm.nih.gov/pubmed/17619723))
