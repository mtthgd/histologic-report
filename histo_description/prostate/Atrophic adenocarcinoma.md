# Atrophic adenocarcinoma

Microscopic (histologic) description

* Well formed, occasionally distorted acini with cells containing scant cytoplasm and infiltrative growth pattern
* Lack of basal cells
* Cells can look flattened or attenuated but mostly preserve malignant cytologic features including
  + Presence of enlarged nucleoli
  + Increased nuclear size
  + Macronucleoli
* Pink amorphous secretions, intraluminal mucin and crystalloids can be seen
