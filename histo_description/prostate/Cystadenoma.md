# Cystadenoma

Microscopic (histologic) description

* Glandular spaces of varying sizes are lined by cuboidal cells without atypia ([Adv Anat Pathol 2015;22:113](https://www.ncbi.nlm.nih.gov/pubmed/25664946))
* Lobular pattern, forming branching lumina and cysts that contain granular intraluminal secretions
* No hypercellularity or atypia in stromal cells
