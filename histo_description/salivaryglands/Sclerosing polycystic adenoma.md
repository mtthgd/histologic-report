# Sclerosing polycystic adenoma

Microscopic (histologic) description

* Circumscribed with preservation of lobular architecture ([Am J Surg Pathol 2006;30:939](https://www.ncbi.nlm.nih.gov/pubmed/16861963))
* Composed of irregularly distributed bilayered ducts and acini within a sclerotic stroma ([Head Neck Pathol 2020;14:630](https://www.ncbi.nlm.nih.gov/pubmed/31605313))
* Ducts range from small tubules to ectatic spaces; may contain secretory material or foamy macrophages ([Head Neck Pathol 2020;14:630](https://www.ncbi.nlm.nih.gov/pubmed/31605313))
  + Ductal cells can have vacuolated, foamy, apocrine and mucous appearances ([Am J Surg Pathol 2006;30:939](https://www.ncbi.nlm.nih.gov/pubmed/16861963))
  + Hyperplastic ductal epithelium forming solid, microcystic and cribriform structures ([Am J Surg Pathol 2017;41:e33](https://www.ncbi.nlm.nih.gov/pubmed/28614209))
  + Small eosinophilic globules of basement membrane-like material in some cribriform areas (i.e. collagenous spherulosis) ([Am J Surg Pathol 1996;20:161](https://www.ncbi.nlm.nih.gov/pubmed/8554105))
* Acini contain numerous, coarse eosinophilic, periodic acid-Schiff positive, intracytoplasmic granules ([Am J Surg Pathol 2006;30:939](https://www.ncbi.nlm.nih.gov/pubmed/16861963))
