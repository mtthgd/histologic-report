# Encapsulated fat necrosis

Microscopic (histologic) description

* Well circumscribed, necrotic adipose tissue encapsulated by
  + Fibrous septa ([Med Ultrason 2020;22:397](https://www.ncbi.nlm.nih.gov/pubmed/32898210))
  + Thin connective tissue capsule ([J Cutan Pathol 2000;27:19](https://www.ncbi.nlm.nih.gov/pubmed/10660127))
* Types of necrosis include ([J Cutan Pathol 2000;27:19](https://www.ncbi.nlm.nih.gov/pubmed/10660127))
  + Macro or microcystic
  + Coagulation
  + Membranous
* Thickness of the fibrous cap is variable ([Br J Plast Surg 2001;54:643](https://www.ncbi.nlm.nih.gov/pubmed/11583506))
* Late stage lesions may demonstrate lymphocytic, histocytic and lipophagic inflammation and possible dystrophic calcification of fibrous cap ([Br J Plast Surg 2001;54:643](https://www.ncbi.nlm.nih.gov/pubmed/11583506), [J Cutan Pathol 2000;27:19](https://www.ncbi.nlm.nih.gov/pubmed/10660127))
