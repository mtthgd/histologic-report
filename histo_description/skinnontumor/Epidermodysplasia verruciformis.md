# Epidermodysplasia verruciformis

Microscopic (histologic) description

* Classic features are mild to moderate acanthosis and hyperkeratosis
* Large cells with blue-gray cytoplasm, perinuclear halos, often dysplastic changes, particularly post kidney transplantation ([J Cutan Pathol 2002;29:480](https://www.ncbi.nlm.nih.gov/pubmed/12207742))
* More commonly has irregular granular layer with rare perinuclear halos
