Microscopic (histologic) description
  • Variable histologic changes nonspecific to any one nutritional deficiency
    • Hyperkeratosis, parakeratosis
    • Keratinocyte pallor and ballooning of upper half of epidermis
  • Vitamin A deficiency (phrynoderma)
    • Hyperkeratosis
    • Keratotic follicular plugging (Indian J Dermatol 2011;56:389)
    • Atrophic sweat glands; squamous metaplasia in severe cases
  • Vitamin C deficiency
    • Extravasated erythrocytes surrounding vessels in upper dermis; may also see perifollicularly (Pediatr Rheumatol Online J 2024;22:56)
    • Coiled, corkscrew-like hairs within keratotic follicular material
