# Scabies mite

Microscopic (histologic) description

* Classic scabies
  + Identification of intracorneal mites, ova or scybala (feces)
    - Mites: 0.2 - 0.4 mm length ([Dermatol Ther 2020;33:e13746](https://www.ncbi.nlm.nih.gov/pubmed/32484302), [Clin Microbiol Rev 2014;27:48](https://www.ncbi.nlm.nih.gov/pubmed/24396136))
    - Oval bodies with rudimentary legs ([Clin Microbiol Rev 2014;27:48](https://www.ncbi.nlm.nih.gov/pubmed/24396136))
    - Exoskeleton with striations, dorsal spines ([Clin Microbiol Rev 2014;27:48](https://www.ncbi.nlm.nih.gov/pubmed/24396136))
    - Long hair-like (setae) on hind legs ([Clin Microbiol Rev 2014;27:48](https://www.ncbi.nlm.nih.gov/pubmed/24396136))
    - Oval eggs: 0.1 - 0.2 mm ([Clin Microbiol Rev 2014;27:48](https://www.ncbi.nlm.nih.gov/pubmed/24396136))
  + Superficial perivascular lymphoeosinophilic infiltrate with histiocytes and possible neutrophils
  + Possible epidermal spongiosis with or without features of excoriation (ulceration, erosion) and crust
  + CD30+ infiltrate possible
* Crusted scabies
  + Hyperkeratosis with numerous mites, eggs and scybala
* Nodular scabies
  + Epidermal acanthosis
  + Denser dermal infiltrate: lymphocytes with or without follicle centers, eosinophils, plasma cells ([Indian J Dermatol Venereol Leprol 1997;63:170](https://www.ncbi.nlm.nih.gov/pubmed/20944314))
  + Vasculitis possible
* Bullous scabies
  + Subepidermal bulla with eosinophils
