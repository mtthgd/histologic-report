Microscopic (histologic) description
  • Prototypically described as having junctional nevic components overlying a scar (Arch Pathol Lab Med 2011;135:842)
  • Melanocytes can exhibit cytologic atypia (Mod Pathol 2009;22:611)
  • Mitotic activity is rarely observed (An Bras Dermatol 2017;92:531, Mod Pathol 2009;22:611, Arch Pathol Lab Med 2011;135:842)
  • Has been observed to display architectural asymmetry and confluent growth (J Cutan Pathol 2001;28:400)
