# Recurrent nevus

Microscopic (histologic) description

* Prototypically described as having junctional nevic components overlying a scar ([Arch Pathol Lab Med 2011;135:842](https://www.ncbi.nlm.nih.gov/pubmed/21732772))
* Melanocytes can exhibit cytologic atypia ([Mod Pathol 2009;22:611](https://www.ncbi.nlm.nih.gov/pubmed/19270643))
* Mitotic activity is rarely observed ([An Bras Dermatol 2017;92:531](https://www.ncbi.nlm.nih.gov/pubmed/28954104), [Mod Pathol 2009;22:611](https://www.ncbi.nlm.nih.gov/pubmed/19270643), [Arch Pathol Lab Med 2011;135:842](https://www.ncbi.nlm.nih.gov/pubmed/21732772))
* Has been observed to display architectural asymmetry and confluent growth ([J Cutan Pathol 2001;28:400](https://www.ncbi.nlm.nih.gov/pubmed/11493377))
