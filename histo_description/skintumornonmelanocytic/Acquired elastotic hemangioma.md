# Acquired elastotic hemangioma

Microscopic (histologic) description

* Broad, band-like proliferation of capillary sized blood vessels arranged horizontally along the superficial dermis, parallel to the epidermis, with prominent associated solar elastosis ([Patterson: Weedon’s Skin Pathology, 5th Edition, 2020](https://www.amazon.com/exec/obidos/ASIN/0702075825/pathologyoutl-20))
* Lacks atypia, mitotic figures, infiltrative growth pattern, deep extension ([J Am Acad Dermatol 2002;47:371](https://www.ncbi.nlm.nih.gov/pubmed/12196746))
