# Dermatomyofibroma

Microscopic (histologic) description

* Somewhat ill defined proliferation of moderately cellular, bland spindled cells arranged in fascicles parallel to the epidermis ([J Cutan Pathol 1992;19:85](https://www.ncbi.nlm.nih.gov/pubmed/1597573), [J Cutan Pathol 2011;38:967](https://www.ncbi.nlm.nih.gov/pubmed/21752049))
* No cytologic atypia or mitoses should be present
* Adnexal structures appear trapped but spared ([J Cutan Pathol 2011;38:967](https://www.ncbi.nlm.nih.gov/pubmed/21752049))
* Bulk of neoplasm is present within the dermis
* Tumor may brush the subcutis but overt involvement is unusual ([Am J Dermatopathol 2009;31:44](https://www.ncbi.nlm.nih.gov/pubmed/19155724))
