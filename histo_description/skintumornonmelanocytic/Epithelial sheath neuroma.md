# Epithelial sheath neuroma

Microscopic (histologic) description

* Architecture
  + Ill defined, located in the papillary dermis, superficial reticular dermis and around follicles ([Am J Surg Pathol 2001;25:696](https://www.ncbi.nlm.nih.gov/pubmed/11342788), [J Cutan Pathol 2009;36:937](https://www.ncbi.nlm.nih.gov/pubmed/19210583))
  + Plaque-like growth pattern ([Am J Surg Pathol 2001;25:696](https://www.ncbi.nlm.nih.gov/pubmed/11342788))
* Cytological features
  + Multiple enlarged peripheral nerve fibers surrounded by mature squamous epithelium ([Arch Toxicol 2011;85:1441](https://www.ncbi.nlm.nih.gov/pubmed/21340527))
  + Nerve bundles contain axons, Schwann cells and perineural cells ([Am J Surg Pathol 2001;25:696](https://www.ncbi.nlm.nih.gov/pubmed/11342788))
  + Squamous sheath contains mature squamous epithelium and may have dyskeratotic keratinocytes
  + Keratinocytes typically do not show atypia, pleomorphism or mitoses ([Clin Neuropathol 2020;39:232](https://www.ncbi.nlm.nih.gov/pubmed/32383640))
  + Neuroepithelial bundles may be surrounded by myxoid stroma and lymphocytic infiltrates
