# Large cell acanthoma

Microscopic (histologic) description

* Sharply demarcated lesion from adjacent normal epidermis ([J Cutan Pathol 2014;41:733](https://www.ncbi.nlm.nih.gov/pubmed/24917472))
* Large cell acanthoma is characterized by
  + Mild acanthosis ([Am J Dermatopathol 1992;14:140](https://www.ncbi.nlm.nih.gov/pubmed/1566974))
  + Epidermal flattening
  + Basketweave hyperkeratosis
  + Enlarged polygonal keratinocytes ([Ocul Oncol Pathol 2019;5:312](https://www.ncbi.nlm.nih.gov/pubmed/31559241))
  + Hypergranulosis ([Am J Dermatopathol 1992;14:140](https://www.ncbi.nlm.nih.gov/pubmed/1566974))
* Characteristics of the nucleus of the enlarged keratinocytes
  + Twice the size of a normal keratinocyte ([J Cutan Pathol 2014;41:733](https://www.ncbi.nlm.nih.gov/pubmed/24917472))
  + Smooth and round nuclear contours ([Am J Dermatopathol 1992;14:140](https://www.ncbi.nlm.nih.gov/pubmed/1566974))
  + Prominent nucleoli ([Ocul Oncol Pathol 2019;5:312](https://www.ncbi.nlm.nih.gov/pubmed/31559241))
  + Lack of nuclear pleomorphism
  + Rare mitotic figures
* Basal layer hyperpigmentation is frequently observed ([Ocul Oncol Pathol 2019;5:312](https://www.ncbi.nlm.nih.gov/pubmed/31559241))
