# Onychopapilloma

Microscopic (histologic) description

* On microscopic examination, onychopapilloma is characterized by
  + Papillomatous proliferation of epithelial cells (82.4% of cases), ovoid bland nuclei
  + Acanthosis (82.4%) which is partially due to the increased thickness of stratum spinosum
  + Matrix metaplasia of nail bed epithelium (41.2%)
  + Hyperkeratosis (29.4%)
  + Distal premature keratinization (70.6%)
  + Dilated vessels resulting in splinter hemorrhage (23.5%) ([J Cutan Pathol 2022;49:147](https://www.ncbi.nlm.nih.gov/pubmed/34396563))
