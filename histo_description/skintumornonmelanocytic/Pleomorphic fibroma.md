# Pleomorphic fibroma

Microscopic (histologic) description

* Resembles fibroepithelial polyp but with enlarged, bizarre, smudged, hyperchromatic nuclei, thick collagen bundles and rare mitotic figures; may be sclerotic ([Am J Dermatopathol 2002;24:54](https://www.ncbi.nlm.nih.gov/pubmed/11803283)) or have myxoid foci ([Am J Dermatopathol 1998;20:502](https://www.ncbi.nlm.nih.gov/pubmed/9790115))
