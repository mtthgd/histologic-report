# Common variable immunodeficiency syndrome

Microscopic (histologic) description

* Mucosa may resemble celiac sprue or be normal but always has reduced plasma cells and no IgA plasma cells
* May have lymphoid hyperplasia or apoptotic bodies in crypts
* Histologic patterns: lymphocytic colitis, collagenous enterocolitis, celiac disease, granulomatous disease, acute GVHD and IBD ([Am J Surg Pathol 2007;31:1800](https://www.ncbi.nlm.nih.gov/pubmed/18043034))
