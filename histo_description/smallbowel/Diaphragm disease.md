# Diaphragm disease

Microscopic (histologic) description

* Thin circumferential membranes resembling plica circularis, composed of mucosa and submucosa with accompanying fibrosis
* Neuromuscular and vascular hamartoma-like changes, eosinophilic enteritis and acute inflammation ([Am J Clin Pathol 2008;130:518](https://www.ncbi.nlm.nih.gov/pubmed/18794043))
