# Necrotizing enterocolitis

Microscopic (histologic) description

* Early: mucosal edema, hemorrhage, necrosis
* Late: hemorrhagic and gangrenous bowel wall, fibrous strictures; often pneumatosis cystoides intestinalis
* After recovery, Paneth cell hyperplasia; colon also shows metaplastic Paneth cells ([Pediatr Res 2011;69:217](https://www.ncbi.nlm.nih.gov/pubmed/21372757))
