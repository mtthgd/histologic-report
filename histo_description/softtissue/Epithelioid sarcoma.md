# Epithelioid sarcoma

Microscopic (histologic) description

* Classic and proximal epithelioid sarcoma can each occur in either proximal or distal sites and may show features of both types ([Histopathology 2001;39:641](https://www.ncbi.nlm.nih.gov/pubmed/11903586))
* Classic or conventional, distal type ([Adv Anat Pathol 2006;13:114](https://www.ncbi.nlm.nih.gov/pubmed/16778474))
  + Dermal / subcutaneous nodule(s)
  + Can simulate granulomatous process, with or without necrosis (pseudogranulomatous morphology)
  + Fairly uniform plump small to medium sized cells with eosinophilic cytoplasm ([Adv Anat Pathol 2016;23:41](https://www.ncbi.nlm.nih.gov/pubmed/26645461))
  + Spindle cells can also be identified and often appear more conspicuous at the periphery of a nodule
  + Mildly atypical nuclei with vesicular chromatin and small nucleoli
  + Variable mitotic activity
  + Mixed chronic inflammatory infiltrate common
  + Dystrophic calcification and metaplastic bone formation
  + Other variant morphologies ([Adv Anat Pathol 2016;23:41](https://www.ncbi.nlm.nih.gov/pubmed/26645461))
    - Angiomatoid variant
    - Myxoid variant
    - Other features include intracytoplasmic vacuoles, multinucleated giant cells and storiform pattern
* Proximal, large cell type ([Am J Surg Pathol 1997;21:130](https://www.ncbi.nlm.nih.gov/pubmed/9042279), [Adv Anat Pathol 2016;23:41](https://www.ncbi.nlm.nih.gov/pubmed/26645461))
  + Infiltrative growth pattern with less defined peripheral borders
  + Multiple large nodules
  + Large polygonal cells with abundant eosinophilic cytoplasm
  + Rhabdoid morphology common
  + Vesicular nuclei with prominent macronucleoli
  + Mitoses, necrosis and hemorrhage common
