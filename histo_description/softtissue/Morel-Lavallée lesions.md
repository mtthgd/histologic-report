# Morel-Lavallée lesions

Microscopic (histologic) description

* Prominent reactive fibroconnective tissue with reactive myofibroblasts
* Fat necrosis and prominent fibrin deposition
* Pseudocystic or cystic component
* Fibrous capsule
* Calcifications, hemosiderin deposition, macrophages
* Mellado and Bencardino classification into 6 subtypes ([Magn Reson Imaging Clin N Am 2005;13:775](https://www.ncbi.nlm.nih.gov/pubmed/16275583))
  + Type I is a serohematic effusion (seroma)
  + Type II is a subacute hematoma
  + Type III is a chronic organizing hematoma
  + Type IV represents perifascial dissection with closed fatty laceration
  + Type V is a perifascial pseudonodular lesion
  + Type VI represents an infected lesion with variable sinus tract formation, internal septations and thick enhancing capsule
* Reference: [Pathology 2018;50:693](https://www.ncbi.nlm.nih.gov/pubmed/30145069)
