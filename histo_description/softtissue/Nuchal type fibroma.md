# Nuchal type fibroma

Microscopic (histologic) description

* Markedly hypocellular
  + Bland spindled fibroblastic cells lining dense cords of collagen
* Thick haphazard collagen bundles
  + Cracked mud pattern: clefts separate bundles
* Entrapment of native structures
  + Adipose tissue
  + Skeletal muscle
  + Nerves
    - Proliferative nerve twigs are sometimes present
    - Similar appearance to peripheral traumatic neuroma
  + Deep skin adnexa
* Unencapsulated
  + Irregular or infiltrative border
* Scant elastic fibers on elastic stain (EVG) ([Skeletal Radiol 2022;51:681](https://www.ncbi.nlm.nih.gov/pubmed/34554278))
* Scattered lymphocytes (variably present)
