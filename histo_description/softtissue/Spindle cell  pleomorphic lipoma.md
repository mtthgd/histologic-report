# Spindle cell  pleomorphic lipoma

Microscopic (histologic) description

* Triad of mature adipocytes, bland spindle cells and hyalinized rope-like collagen fibers
* Multinucleated stromal giant cells and floret cells in pleomorphic lipoma
* Myxoid background with mast cells frequent
* Short and stubby spindle cells show occasional palisading
* Rare lipoblasts present in half of cases ([Hum Pathol 2017;65:140](https://www.ncbi.nlm.nih.gov/pubmed/28546131))
* Infiltrative growth, necrosis, atypical spindle cells, pleomorphic lipoblasts and significant mitotic activity usually absent ([Semin Diagn Pathol 2019;36:105](https://www.ncbi.nlm.nih.gov/pubmed/30850230))
* Fat poor and fat free subtype: contains little (Am J Dermatopathol 2007;29:437, [Am J Dermatopathol 2009;31:423](https://www.ncbi.nlm.nih.gov/pubmed/19542913))
* Myxoid subtype: contains abundant myxoid stroma ([Pathol Int 2014;64:346](https://www.ncbi.nlm.nih.gov/pubmed/25047505))
* (Pseudo) angiomatous subtype: contains dilated endothelial lined channels and nodules of floating tumor ([Histopathology 1994;24:565](https://www.ncbi.nlm.nih.gov/pubmed/8063285), [Pathol Int 2007;57:26](https://www.ncbi.nlm.nih.gov/pubmed/17199739))
* Plexiform variant has been described ([Histopathology 1995;27:533](https://www.ncbi.nlm.nih.gov/pubmed/8838333))
* Focal cartilaginous and osseous metaplasia rarely present ([Histopathology 1987;11:803](https://www.ncbi.nlm.nih.gov/pubmed/3623439))
* Extramedullary hematopoiesis may occur ([Histopathology 2001;39:215](https://www.ncbi.nlm.nih.gov/pubmed/11493343))
