# Undifferentiated  unclassified sarcoma

Microscopic (histologic) description

* Storiform, fascicular or patternless arrangement of highly atypical spindled, epithelioid, round or pleomorphic cells ([Lindberg: Diagnostic Pathology - Soft Tissue Tumors, 2nd Edition, 2015](https://www.amazon.com/exec/obidos/ASIN/0323376754/pathologyoutl-20), [Ultrastruct Pathol 2008;32:31](https://www.ncbi.nlm.nih.gov/pubmed/18446665), [Mod Pathol 2014;27:S39](https://www.ncbi.nlm.nih.gov/pubmed/24384852))
  + Undifferentiated round cell sarcoma can be diagnosed only if distinctive gene fusions have been ruled out
* Fascicular or patternless architecture
* Frequent bizarre multinucleated tumor giant cells
* Amphophilic or palely eosinophilic cytoplasm
* Hyperchromatic or vesicular nuclei
* Abundant mitoses
* Coagulative necrosis is common
