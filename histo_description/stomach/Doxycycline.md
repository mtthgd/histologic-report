# Doxycycline

Microscopic (histologic) description

* Superficial mucosal capillaries with eosinophilic degenerative changes ([Am J Surg Pathol 2013; 37:259](https://www.ncbi.nlm.nih.gov/pubmed/23060354))
* Background gastric mucosa with reactive changes in foveolar epithelium, superficial mucosal necrosis and neutrophilic infiltration within lamina propria
