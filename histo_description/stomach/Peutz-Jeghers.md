# Peutz-Jeghers

Microscopic (histologic) description

* Core of finely arborizing branches of smooth muscle from muscularis mucosa
* Covered by normal but often disorganized mucosa
* Usually no prominent inflammation
* Pseudoinvasion in 10% (no atypia, normal epithelial cell subtypes, brush border, hemosiderin deposition, intramural mucinous cysts, [Am J Surg Pathol 1987;11:743](https://www.ncbi.nlm.nih.gov/pubmed/3661821))
