# Plexiform fibromyxoma

Microscopic (histologic) description

* Multinodular proliferation of bland spindle cells and arborizing capillaries in a myxoid stroma ([World J Gastroenterol 2010;16:2835](https://www.ncbi.nlm.nih.gov/pubmed/20556828))
* Variable cellularity
* Rare or no mitoses
* Some degree of atypia and slightly increased mitotic activity rarely seen ([J Surg Res 2019;239:76](https://www.ncbi.nlm.nih.gov/pubmed/30822694))
