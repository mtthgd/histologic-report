# Xanthoma

Microscopic (histologic) description

* Histologic hallmark is the presence of lipid laden foamy macrophages (xanthoma cells) within the lamina propria ([Ann Diagn Pathol 2013;17:72](https://www.ncbi.nlm.nih.gov/pubmed/23020998), [Arch Pathol Lab Med 2004;128:937](https://www.ncbi.nlm.nih.gov/pubmed/15270598))
