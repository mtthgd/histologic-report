# Intratubular large cell hyalinizing Sertoli cell neoplasia ITLCHSCN

Microscopic (histologic) description

* Patchy, intratubular growth pattern ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
* Expanded seminiferous tubules, 2 - 4x the normal size ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
* Proliferation of enlarged Sertoli cells ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
  + Abundant cytoplasm, pale to eosinophilic, often vacuolated
  + Uniform nuclei, round to oval
  + Fine chromatin
  + Small to medium nucleoli
* Thickened peritubular basement membrane ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
* Projections of basement membrane forming intratubular globular eosinophilic deposits ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
* Spermatogonia not present ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
* Rare, small focal calcifications in basement membrane deposits ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
* No mitotic figures ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
* Nonneoplastic tubules with prepubertal histological features ([Am J Surg Pathol 2007;31:827](https://www.ncbi.nlm.nih.gov/pubmed/17527069))
