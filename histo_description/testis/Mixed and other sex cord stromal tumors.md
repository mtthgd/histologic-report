# Mixed and other sex cord stromal tumors

Microscopic (histologic) description

* Mixture of cell types granulosa cell tumor, Leydig cell tumor, thecoma, etc
* One particular cell type may predominate commonly the spindle cell type ([Mod Pathol 1997;10:693](https://www.ncbi.nlm.nih.gov/pubmed/9237180), [Pathol Res Pract 2007;203:759](https://www.ncbi.nlm.nih.gov/pubmed/17850986))
