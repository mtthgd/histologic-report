# Testicular regression syndrome  anorchia

Microscopic (histologic) description

* Regressed testis indicated by fibrosis, hemosiderin, calcification or Leydig cells near epididymis or proximal vas deferens ([Arch Pathol Lab Med 2000;124:694](https://www.ncbi.nlm.nih.gov/pubmed/10782149))
* Presence of only fat and connective tissue does not rule out an intraabdominal testis
