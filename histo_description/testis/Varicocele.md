# Varicocele

Microscopic (histologic) description

* From inside to outside, changes include ([Ultrastruct Pathol 2010;34:260](https://www.ncbi.nlm.nih.gov/pubmed/20929307))
  + Narrowing or obliteration of the vein lumens
  + Segmental obliteration and occasional thrombi
  + Internal elastic lamina fragmentation and invagination of intima
  + Variable thickening of vein wall; the media contains hypertrophied smooth muscle fibers and deposition of collagen bundles ([Ultrastruct Pathol 2012;36:201](https://www.ncbi.nlm.nih.gov/pubmed/22849521))
* If a testis is included: decreased spermatogenesis in tubules with germ cell degeneration and increased Leydig cells
* Adolescent varicocele: pathologic changes found at or soon after puberty, consisting of tubular sclerosis, premature germ cell sloughing, small vessel sclerosis and variable hypospermatogenesis ([Am J Clin Pathol 1988;89:321](https://www.ncbi.nlm.nih.gov/pubmed/3348167))
