# Black  pigmented thyroid

Microscopic (histologic) description

* Dark brown, granular, dust-like cytoplasmic pigment abundant in apical portion of follicular cells, colloid and stromal macrophages ([Am J Pathol 1984;117:98](https://www.ncbi.nlm.nih.gov/pubmed/6435454))
* Pigment is nonfluorescent and nonbirefringent ([Arch Pathol Lab Med 2004;128:355](https://www.ncbi.nlm.nih.gov/pubmed/14987144))
