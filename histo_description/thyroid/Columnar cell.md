# Columnar cell

Microscopic (histologic) description

* Patterns
  + Papillary is the most common ([Cancer Cytopathol 2017;125:389](https://www.ncbi.nlm.nih.gov/pubmed/28374549))
  + Others include trabecular, follicular, solid and cribriform ([Histopathology 2018;72:40](https://www.ncbi.nlm.nih.gov/pubmed/29239042), [Turk Patoloji Derg 2015;31:34](https://www.ncbi.nlm.nih.gov/pubmed/26177316), [Mod Pathol 2011;24:739](https://www.ncbi.nlm.nih.gov/pubmed/21358618))
  + Rare squamous morule formation or rosette-like structures ([Mod Pathol 2011;24:739](https://www.ncbi.nlm.nih.gov/pubmed/21358618), [Cytojournal 2014;11:27](https://www.ncbi.nlm.nih.gov/pubmed/25379049))
* Columnar cells
  + Clear to pale eosinophilic cytoplasm ([Mod Pathol 2011;24:739](https://www.ncbi.nlm.nih.gov/pubmed/21358618))
  + Supranuclear or subnuclear cytoplasmic vacuoles resembling secretory endometrium ([Histopathology 2018;72:40](https://www.ncbi.nlm.nih.gov/pubmed/29239042), [Adv Anat Pathol 2018;25:172](https://www.ncbi.nlm.nih.gov/pubmed/29351089), [Cancer Cytopathol 2017;125:389](https://www.ncbi.nlm.nih.gov/pubmed/28374549))
* Nuclear features
  + Prominent pseudostratification ([Cancer Cytopathol 2017;125:389](https://www.ncbi.nlm.nih.gov/pubmed/28374549))
  + Elongated, hyperchromatic nuclei with coarse chromatin ([Histopathology 2018;72:40](https://www.ncbi.nlm.nih.gov/pubmed/29239042), [Adv Anat Pathol 2018;25:172](https://www.ncbi.nlm.nih.gov/pubmed/29351089), [Head Neck 2011;33:1052](https://www.ncbi.nlm.nih.gov/pubmed/20824810))
  + Rare or absent chromatin clearing, nuclear grooves and intranuclear inclusions
  + Nuclear grooves may be prominent ([Cancer Cytopathol 2017;125:389](https://www.ncbi.nlm.nih.gov/pubmed/28374549), [Cytojournal 2014;11:27](https://www.ncbi.nlm.nih.gov/pubmed/25379049))
  + Inconspicuous nucleoli
* Mitotic activity can be increased ([Adv Anat Pathol 2018;25:172](https://www.ncbi.nlm.nih.gov/pubmed/29351089), [Mod Pathol 2011;24:739](https://www.ncbi.nlm.nih.gov/pubmed/21358618))
* Variable necrosis ([Mod Pathol 2011;24:739](https://www.ncbi.nlm.nih.gov/pubmed/21358618))
* Usually lacks psammoma bodies ([Adv Anat Pathol 2018;25:172](https://www.ncbi.nlm.nih.gov/pubmed/29351089), [Cancer Cytopathol 2017;125:389](https://www.ncbi.nlm.nih.gov/pubmed/28374549))
