# DiGeorge syndrome

Microscopic (histologic) description

* Consistent with normal thyroid histology (most specimens come from pediatric autopsies)
* Rarely, pathological changes has been described:
  + Predominant fetal follicles
  + Hyperplasia
  + Follicular adenoma and nodular goiter ([Case Rep Med 2013;2013:923129](https://www.ncbi.nlm.nih.gov/pubmed/23762078))
  + Thyroid carcinoma ([Am J Med Genet A 2006;140:906](https://www.ncbi.nlm.nih.gov/pubmed/16532473))
* C cells can be absent or significantly reduced in number, consistently with their pharyngeal pouch origin ([Hum Pathol 1987;18:355](https://www.ncbi.nlm.nih.gov/pubmed/3557439), [Am J Med Genet 1993;46:641](https://www.ncbi.nlm.nih.gov/pubmed/8362905), [Pediatr Pathol 1993;13:463](https://www.ncbi.nlm.nih.gov/pubmed/8372031))
