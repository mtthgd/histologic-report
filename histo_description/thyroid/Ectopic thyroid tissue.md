# Ectopic thyroid tissue

Microscopic (histologic) description

* The same histological appearance as the orthotopic thyroid tissue, composed of colloid filled follicles
* Signs of hyperplasia or thyroiditis in relevant clinical situations
* Ectopic thyroid tissue showing cytologic or histologic features of thyroid carcinoma should be considered as metastases until proven otherwise, although there are several reports of tumors arising de novo in ectopic thyroid ([Thyroid 2007;17:603](https://www.ncbi.nlm.nih.gov/pubmed/17696827)):
  + Follicular adenoma ([J Med Case Rep 2014;8:270](https://www.ncbi.nlm.nih.gov/pubmed/25103611))
  + Follicular carcinoma ([Am J Clin Pathol 1987;87:776](https://www.ncbi.nlm.nih.gov/pubmed/3591751))
  + Classic variant of papillary thyroid carcinoma ([BMJ Case Rep. 2012 Jul 9;2012](https://www.ncbi.nlm.nih.gov/pubmed/22778453))
  + Follicular variant of papillary thyroid carcinoma ([Ear Nose Throat J 2009;88:E7](https://www.ncbi.nlm.nih.gov/pubmed/19517394))
  + Hurthle cell carcinoma ([Acta Cytol 1983;27:188](https://www.ncbi.nlm.nih.gov/pubmed/6573091))
  + Medullary thyroid carcinoma ([Singapore Med J 2008;49:251](https://www.ncbi.nlm.nih.gov/pubmed/18363009))
  + Poorly differentiated thyroid carcinoma ([Endocr Pathol 2002;13:353](https://www.ncbi.nlm.nih.gov/pubmed/12665653))
  + Anaplastic carcinoma ([BMJ Case Rep 2010 Aug 31; 2010](http://casereports.bmj.com/content/2010/bcr.03.2010.2823.abstract))
  + Teratoma ([Anal Quant Cytol Histol 2009;31:233](https://www.ncbi.nlm.nih.gov/pubmed/19736871))
  + B cell lymphoma ([Acta Chir Belg 2009;109:802](https://www.ncbi.nlm.nih.gov/pubmed/20184075))
