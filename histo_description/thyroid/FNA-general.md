# FNA-general

Microscopic (histologic) description

* Histopathological changes in thyroid after FNA procedure include acute and chronic:
  + Acute changes are hemorrhage, granulation tissue, giant cells, hemosiderin laden macrophages, cholesterol clefts, necrosis, rarely infarction ([Diagn Cytopathol 1996;15:211](https://www.ncbi.nlm.nih.gov/pubmed/8955603))
  + Chronic changes are infarction, oncocytes, squamous or spindle metaplasia, fibrosis, calcification, vascular thrombosis; rarely tumor implantation along needle tract, rarely papillary endothelial hyperplasia ([World J Surg 2020;44:378](https://www.ncbi.nlm.nih.gov/pubmed/31515571), [Pathol Res Pract 2007;203:641](https://www.ncbi.nlm.nih.gov/pubmed/17582696), [Arch Pathol Lab Med 1992;116:1040](https://www.ncbi.nlm.nih.gov/pubmed/1417444), [Arch Pathol Lab Med 1991;115:240](https://www.ncbi.nlm.nih.gov/pubmed/2001162))
