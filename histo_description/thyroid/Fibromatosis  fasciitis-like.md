# Fibromatosis  fasciitis-like

Microscopic (histologic) description

* Mixed epithelial-mesenchymal tumor
* Epithelial component:
  + PTC (usually classical PTC; follicular variant of PTC also documented) ([Mod Pathol 2020;33:1702](https://www.ncbi.nlm.nih.gov/pubmed/ft32291398))
* Mesenchymal component:
  + 20 - 95% of tumor area ([Endocr J 2017;64:1017](https://www.ncbi.nlm.nih.gov/pubmed/ft28794344), [Ultrasound Int Open 2018;4:E39](https://www.ncbi.nlm.nih.gov/pubmed/ft30250939))
  + Myofibroblastic proliferation similar to fibromatosis / nodular fasciitis / other myofibroblastic proliferative processes
  + Fibromatosis type stroma: dense hypocellular collagenous connective tissue
  + Nodular fasciitis-like stroma: edematous, extravasated red blood cells, inflammatory cells; reminiscent of granulation tissue
  + May infiltrate surrounding neck structures
  + May metastasize to lymph nodes along with the epithelial cell component ([Endocr J 2017;64:1017](https://www.ncbi.nlm.nih.gov/pubmed/ft28794344))
