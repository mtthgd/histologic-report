# Hyperparathyroidism

Microscopic (histologic) description

* Osteoblastic and osteoclastic activity, cysts, hemosiderin laden macrophages
* Pale, vacuolated cells arranged in a trabecular pattern are also seen in non-PTH mediated hypercalcemia
  ([Am J Surg Pathol 1985;9:43](https://www.ncbi.nlm.nih.gov/pubmed/3970297))
