# Hypothyroidism

Microscopic (histologic) description

* Primary hypothyroidism ([J Clin Endocrinol Metab 1993;76:466](https://www.ncbi.nlm.nih.gov/pubmed/8432791)):
  + Hyperplastic follicles, some with papillary folding
  + Follicle cells range from cuboidal to columnar with clear cytoplasm and round nuclei
  + Scant colloid material
  + Lymphocytic infiltrate may be present
* Skin myxedema is usually characterised by ([J Clin Diagn Res 2014;8:YD01](https://www.ncbi.nlm.nih.gov/pubmed/25121051)):
  + Collagen fibers in reticular dermis are separated by mucin accumulation
  + Cutaneous mucinosis ([Am J Dermatopathol 2010;32:196](https://www.ncbi.nlm.nih.gov/pubmed/19940747))
  + Mild perivascular lymphocyte infiltrate
  + Epidermis is spared
  + Prominent fibroblasts ([An Bras Dermatol 2016;91:100](https://www.ncbi.nlm.nih.gov/pubmed/26982789))
  + Presence of hemosiderin deposits ([An Bras Dermatol 2016;91:100](https://www.ncbi.nlm.nih.gov/pubmed/26982789))
