# Parasitic nodule

Microscopic (histologic) description

* Benign appearing thyroid tissue with colloid filled or hyperplastic follicles
* Similar features are found in orthotopic gland
* Hashimoto thyroiditis in parasitic nodule may simulate lymph node tissue
  In addition, reactive atypia in Hashimoto follicular cells with ground glass nuclei mimics papillary thyroid carcinoma ([Rev Hosp Clin Fac Med Sao Paulo 2000;55:65](https://www.ncbi.nlm.nih.gov/pubmed/10959126))
