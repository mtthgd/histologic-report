# SMARCA4 deficient uterine sarcoma

Microscopic (histologic) description

* Small to large epithelioid cells with prominent nucleoli and brisk mitotic activity ([Mod Pathol 2018;31:1442](https://www.ncbi.nlm.nih.gov/pubmed/29700418), [Am J Surg Pathol 2020;44:263](https://www.ncbi.nlm.nih.gov/pubmed/31567195))
* Diffuse growth of tightly packed cells, sometimes in a nested / corded architecture with stromal hyalinization ([Mod Pathol 2018;31:1442](https://www.ncbi.nlm.nih.gov/pubmed/29700418), [Am J Surg Pathol 2020;44:263](https://www.ncbi.nlm.nih.gov/pubmed/31567195))
* Rhabdoid morphology is common ([Mod Pathol 2018;31:1442](https://www.ncbi.nlm.nih.gov/pubmed/29700418), [Am J Surg Pathol 2020;44:263](https://www.ncbi.nlm.nih.gov/pubmed/31567195))
* Necrosis is common ([Mod Pathol 2018;31:1442](https://www.ncbi.nlm.nih.gov/pubmed/29700418), [Am J Surg Pathol 2020;44:263](https://www.ncbi.nlm.nih.gov/pubmed/31567195))
* Lymphovascular invasion is common ([Mod Pathol 2018;31:1442](https://www.ncbi.nlm.nih.gov/pubmed/29700418), [Am J Surg Pathol 2020;44:263](https://www.ncbi.nlm.nih.gov/pubmed/31567195))
* Phyllodiform-like growth may be present ([Am J Surg Pathol 2020;44:263](https://www.ncbi.nlm.nih.gov/pubmed/31567195))
