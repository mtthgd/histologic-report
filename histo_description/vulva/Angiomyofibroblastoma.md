# Angiomyofibroblastoma

Microscopic (histologic) description

* Well circumscribed, noninfiltrative
* Fibrous pseudocapsule in a subset ([Am J Surg Pathol 1992;16:373](https://www.ncbi.nlm.nih.gov/pubmed/1314521))
* Characteristic alternating hypo and hypercellular zones ([Am J Surg Pathol 1992;16:373](https://www.ncbi.nlm.nih.gov/pubmed/1314521))
* Tumor cells:
  + Spindle to epithelioid to plasmacytoid myofibroblastic cells
  + Typically bland chromatin and inconspicuous nucleoli; scattered mildly atypical cells in a minority ([Am J Surg Pathol 1992;16:373](https://www.ncbi.nlm.nih.gov/pubmed/1314521))
  + Bland multinucleated cells common ([Am J Surg Pathol 1992;16:373](https://www.ncbi.nlm.nih.gov/pubmed/1314521))
  + Mitoses rare or absent; no atypical mitoses
* Tumor stroma and vasculature:
  + Lacks prominent stromal mucin
  + Hypocellular foci show edematous stroma with scattered fine to thick bands of stromal collagen
  + In hypocellular foci, tumor cells appear randomly distributed
  + In hypercellular foci, tumor cells congregate around small, irregularly distributed, thin walled vessels
  + Occasional larger vessels interspersed
  + Mast cells typically conspicuous
  + Adipocytic differentiation in ~ 25 - 50% ([Mod Pathol 1996;9:284](https://www.ncbi.nlm.nih.gov/pubmed/8685229), [Hum Pathol 1997;28:1046](https://www.ncbi.nlm.nih.gov/pubmed/9308729))
* Rare cases with predominant adipocytic differentiation = lipomatous variant of angiomyofibroblastoma ([Hum Pathol 1997;28:1046](https://www.ncbi.nlm.nih.gov/pubmed/9308729), [Hum Pathol 2014;45:1647](https://www.ncbi.nlm.nih.gov/pubmed/24880711), [Int J Gynecol Pathol 2015;34:204](https://www.ncbi.nlm.nih.gov/pubmed/25675192))
* Rare tumors show mixed features of angiomyofibroblastoma and aggressive angiomyxoma; should be managed like aggressive angiomyxoma ([Histopathology 1997;30:3](https://www.ncbi.nlm.nih.gov/pubmed/9023551))
* Sole case with sarcomatous transformation showed hypercellular foci with cytologic atypia, increased mitoses and lymphovascular invasion ([Am J Surg Pathol 1997;21:1104](https://www.ncbi.nlm.nih.gov/pubmed/9298888))
