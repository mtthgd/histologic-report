Definition / general

* Heterogenous group of diffuse, histologically high grade gliomas, typically occurring in children, adolescents or young adults, genetically wild type for histone H3, *IDH1* and *IDH2* and showing a distinct clustering pattern on global methylation analysis
* CNS WHO grade 4 of 4

Essential features

* First described in 2017 as a heterogeneous group of pediatric glioblastoma without histone 3 or *IDH* mutations; further classified into RTK1, RTK2 and MYCN subtypes by genome wide methylation profiling ([Acta Neuropathol 2017;134:507](https://www.ncbi.nlm.nih.gov/pubmed/28401334))
  + RTK1 subtype: enriched with *PDGFRA* mutations (33%), lacking significant copy number variations and showing intermediate prognosis (median overall survival [OS]: 21 months)
  + RTK2 subtype: enriched with mutations common in adult type glioblastoma, including *EGFR* amplification (50%), chromosome 7 gain (28%) / 10q loss (50%), *CDKN2A* / *CDKN2B* homozygous deletions (72%) and *TERT* promoter mutations (64%), and showing the best prognosis (median OS: 44 months)
  + MYCN subtype is enriched in *MYCN* (50%) amplification with *ID2* coamplification (66%), *MYC* (8%) amplification and *TP53* mutations (67%); this group has the worst prognosis (median OS: 14 months)
  + Clustered separately from adult glioblastomas by global methylation profiling
* *PDGFRA, EGFR, MYCN* amplifications represent their respective methylation subtypes but are not specific and can cross over into other subtypes (e.g., *MYCN* amplification can be seen in tumors with RTK1 and RTK2 methylation subtypes) ([Acta Neuropathol 2017;134:507](https://www.ncbi.nlm.nih.gov/pubmed/28401334), [Acta Neuropathol Commun 2020;8:104](https://www.ncbi.nlm.nih.gov/pubmed/32646492))
* DNA methylation defined tumor entity based on clustering analyses; diagnosis can only be made with certainty by DNA methylation profiling ([Acta Neuropathol 2017;134:507](https://www.ncbi.nlm.nih.gov/pubmed/28401334), [Acta Neuropathol Commun 2020;8:104](https://www.ncbi.nlm.nih.gov/pubmed/32646492))
* Similar methylation patterns have been identified in adult patients, however, the incidence in adult patients is unclear and likely underestimated ([Brain Tumor Pathol 2023;40:204](https://www.ncbi.nlm.nih.gov/pubmed/37561227))

Terminology

* Pediatric type glioblastoma, H3 wild type (not recommended)
* Methylation class glioblastoma midline (MC GBM MID) (not recommended)
* Pediatric glioblastoma (pedGBM) (not recommended)
* Diffuse high grade glioma in childhood, H3 wild type, group C (pHGG WT C) (not recommended)
* Methylation class glioblastoma RTK3 (MC GBM RTK III) (not recommended)
* Diffuse high grade glioma in childhood, H3 wild type, group B (pHGG WT B) (not recommended)
* Methylation class glioblastoma MYCN (MC GBM MYCN) (not recommended)

ICD coding

* ICD-O: [9385/3](https://academic.oup.com/neuro-oncology/article/24/11/1811/6649746#jumplink-T1) - diffuse pediatric type high grade glioma, H3 wild type and *IDH* wild type
* ICD-10: [C71.9](https://www.icd10data.com/ICD10CM/Codes/C00-D49/C69-C72/C71-/C71.9) - malignant neoplasm of brain, unspecified
* ICD-11: [2A00.0Y & XH4Q01](https://icd.who.int/browse/2024-01/mms/en#422307135%2Fother%20%26%20150565601) - other specified gliomas of brain & diffuse pediatric type high grade glioma, H3 wild type and *IDH* wild type

Epidemiology

* Age: initially described in children, adolescents and young adults ([Brain Tumor Pathol 2023;40:204](https://www.ncbi.nlm.nih.gov/pubmed/37561227))
* Recently reported in adults and in the elderly, although incidence in adults is unclear and likely underestimated ([Brain Tumor Pathol 2023;40:204](https://www.ncbi.nlm.nih.gov/pubmed/37561227))

Sites

* Most common: supratentorial (82 - 96%)
* Less common: infratentorial (4 - 18%)
* Subtype specific incidence ([Acta Neuropathol 2017;134:507](https://www.ncbi.nlm.nih.gov/pubmed/28401334))
  + RTK1 subtype: supratentorial = 82%, infratentorial = 18%
  + RTK2 subtype: supratentorial = 96%, infratentorial = 4%
  + MYCN subtype: supratentorial = 86%, infratentorial = 14%

Pathophysiology

* RTK / RAS / PI3K pathway is commonly altered, particularly through *EGFR* or *PDGFRA* amplification
* Loss of function in tumor suppressor pathways, such as the p53 pathway (via *TP53* mutations) or RB pathway (via *CDKN2A* / *CDKN2B* deletions), is also implicated
* Epigenetic changes: abnormal characteristic DNA methylation increasingly recognized as a contributor ([Acta Neuropathol 2017;134:507](https://www.ncbi.nlm.nih.gov/pubmed/28401334), [Acta Neuropathol Commun 2024;12:57](https://www.ncbi.nlm.nih.gov/pubmed/38605367))

Etiology

* Therapeutic radiation, germline mismatch repair deficiency for diffuse pediatric type high grade glioma (pHGG), RTK1 subtype ([Acta Neuropathol 2019;137:139](https://www.ncbi.nlm.nih.gov/pubmed/30196423), [Acta Neuropathol Commun 2024;12:57](https://www.ncbi.nlm.nih.gov/pubmed/38605367))
* Li-Fraumeni syndrome (germline *TP53* mutation) for pHGG MYCN subtype ([Acta Neuropathol Commun 2023;11:3](https://www.ncbi.nlm.nih.gov/pubmed/36609284), [Acta Neuropathol Commun 2023;11:170](https://www.ncbi.nlm.nih.gov/pubmed/37875973))

Clinical features

* Depends on tumor location ([Brain Tumor Pathol 2023;40:204](https://www.ncbi.nlm.nih.gov/pubmed/37561227))

Diagnosis

* Based on imaging / biopsy / resection specimens

Radiology description

* MRI: T1 hypointense to isointense, with inhomogeneous contrast enhancement, often with rim enhancement; T2 hyperintense, no restriction in diffusion weighted imaging (DWI) ([Brain Tumor Pathol 2023;40:204](https://www.ncbi.nlm.nih.gov/pubmed/37561227))
* Some cases can be nonenhancing (see [case 2 image](https://www.pathologyoutlines.com/topic/CNStumordiffusepediatrichighglioma.html#diffusepediatrichighgliomaradio) below)

Prognostic factors

* Different prognosis according to subtype ([Acta Neuropathol 2017;134:507](https://www.ncbi.nlm.nih.gov/pubmed/28401334))
  + RTK1 subtype: 21 months mean overall survival
  + RTK2 subtype: 44 months (best prognosis) mean overall survival
  + MYCN subtype: 14 months (worst prognosis) mean overall survival

Case reports

* Case series study of 8 cases ranging in age from 8 to 71 years ([Brain Tumor Pathol 2023;40:204](https://www.ncbi.nlm.nih.gov/pubmed/37561227))
* Case series of 3 adult patients with Li-Fraumeni syndrome and pediatric type high grade gliomas ([Acta Neuropathol Commun 2024;12:57](https://www.ncbi.nlm.nih.gov/pubmed/38605367))

Treatment

* Usually includes a combination of surgery, radiotherapy and chemotherapy depending on suitability
* Multiple novel treatments including targeted therapy, immunotherapy, focused ultrasound and sonodynamic therapy are in clinical trial stages ([Front Oncol 2024;14:1347694](https://www.ncbi.nlm.nih.gov/pubmed/38525424))

Gross description

* Infiltrative lesion with anatomical distortion of the surrounding structures
* Areas of hemorrhage and necrosis may be identified

Microscopic (histologic) description

* pHGG MYCN subtype may demonstrate a small blue cell tumor morphology resembling a primitive neuronal component ([Acta Neuropathol Commun 2019;7:87](https://www.ncbi.nlm.nih.gov/pubmed/31177990), [Acta Neuropathol Commun 2020;8:104](https://www.ncbi.nlm.nih.gov/pubmed/32646492))
  + Olig2 and GFAP expressions may be focal or absent
  + p53 is usually diffuse and strong
* Histology of the RTK1 and RTK2 subtypes is variable and usually demonstrates high grade glioma features, including atypia, mitoses, microvascular proliferation and necrosis ([Acta Neuropathol Commun 2024;12:57](https://www.ncbi.nlm.nih.gov/pubmed/38605367), [Brain Tumor Pathol 2023;40:204](https://www.ncbi.nlm.nih.gov/pubmed/37561227))

Sample pathology report

* Brain, right temporal lobe, resection:
  + Pediatric type diffuse high grade glioma, H3 wild type and *IDH* wild type, subtype RTK1, CNS WHO grade 4
  + Additional molecular genetic information
    - *CDKN2A* / *CDKN2B* homozygous loss
    - Mutations in *NF1, APC* and *ARID1A*
    - Multiple copy number variations (CNVs): including whole arm loss of 9p, 10 and partial loss of 1, 4, 5, 14, 17, 18, 19
    - *MGMT* promoter methylated

Differential diagnosis

* [Infant type hemispheric glioma](https://www.pathologyoutlines.com/topic/CNStumorinfanttype.html):
  + Usually occurs in infants Molecularly characterized by RTK fusions including *NTRK* family genes, *ROS1, ALK* or *MET*
  + Tumor cells are more spindled than round
* [Desmoplastic infantile ganglioglioma / desmoplastic infantile astrocytoma](https://www.pathologyoutlines.com/topic/cnstumordesmoplasticinfantileganglioglioma.html):
  + Usually occurs in infants with a more superficial location and sharp border, with molding of overlying cranial bones
  + Histologically contains a desmoplastic spindle cell component rich in reticulin and a neuroepithelial component
* [Medulloblastoma](https://www.pathologyoutlines.com/topic/cnstumormedulloblastoma.html):
  + Occurs exclusively in cerebellum
  + Has distinct molecular / genetic profile
  + [Olig2](https://www.pathologyoutlines.com/topic/stainsolig2.html) negative
* [Glioblastoma *IDH* wild type](https://www.pathologyoutlines.com/topic/CNStumorgliomasglioblastomasIDHwildtype.html):
  + pHGG H3 wild type, *IDH* wild type predominantly occurs in children, *IDH* wild type glioblastoma predominantly occurs in adults
  + True incidence of pHGG H3 wild type, *IDH* wild type in the adult population is unclear and might be underestimated; the clinical necessity of separating these 2 diagnoses in the adult population is yet to be clarified
  + Morphology and molecular genetic features significantly overlap between pHGG RTK2 subtype and adult *IDH* wild type glioblastoma and can only be differentiated by global methylation analysis
* Other CNS embryonal tumors:
  + [Atypical teratoid / rhabdoid tumor](https://www.pathologyoutlines.com/topic/cnstumoratypicalteratoidrhabdoid.html):
    - Loss of [INI1 / SMARCB1](https://www.pathologyoutlines.com/topic/stainsini1.html) or [SMARCA4](https://www.pathologyoutlines.com/topic/stainsSMARCA4BRG1.html) nuclear expression
  + [CNS neuroblastoma, *FOXR2* activated](https://www.pathologyoutlines.com/topic/cnsneuroblastoma.html):
    - Primitive embryonal appearance with *FOXR2* fusion
    - [Olig2](https://www.pathologyoutlines.com/topic/stainsolig2.html), [TTF1](https://www.pathologyoutlines.com/topic/stainsttf1.html) diffuse and strong, [GFAP](https://www.pathologyoutlines.com/topic/stainsgfap.html) usually negative in tumor cells
  + [CNS tumor with *BCOR* internal tandem duplication](https://www.pathologyoutlines.com/topic/CNStumorBCOR.html):
    - Defined by presence of internal tandem duplication in exon 15 of the *BCOR* gene
    - Diffuse strong nuclear [BCOR](https://www.pathologyoutlines.com/topic/stainsbcor.html) IHC expression
* [Pleomorphic xanthoastrocytoma](https://www.pathologyoutlines.com/topic/cnstumorpleomorphicxanthoastrocytoma.html):
  + Usually contains higher cytologic pleomorphism, eosinophilic granular bodies (EGB) and reticulin rich areas and typically lacks [p53](https://www.pathologyoutlines.com/topic/stainsp53.html) expression
  + Genetically characterized by MAPK alterations (*BRAF* p. V600E mutation) co-occurring with homozygous loss of *CDKN2A* / *CDKN2B*
* [Oligodendroglioma](https://www.pathologyoutlines.com/topic/cnstumoroligodendrogliomaidhmutant.html):
  + By definition, these tumors have *IDH* mutations
* [Diffuse midline glioma, H3 K27 altered](https://www.pathologyoutlines.com/topic/cnstumordmgh3k27m.html):
  + By definition, these tumors have *H3K27* mutations
* [Diffuse midline glioma, H3 G34 mutant](https://www.pathologyoutlines.com/topic/cnstumordiffuseH3G34mutant.html):
  + By definition, these tumors have *H3G34* mutations

Board review style question #1

[![](https://www.pathologyoutlines.com/imgau/CNStumordiffusepediatrichighgliomaMicroElnagdy5.jpg)](https://www.pathologyoutlines.com/imgau/CNStumordiffusepediatrichighgliomaMicroElnagdy5.jpg)

A 43 year old man presents with progressively worsening headache and gait disturbance over several weeks. He is mildly confused and slow to respond. MRI shows a large, right frontal lobe mass in the brain with areas of contrast enhancement. Craniotomy with resection was done and representative H&E section is shown. Tumor cells are negative for IDH1 R132H and show retained ATRX nuclear expression. There is focal positivity for GFAP and widespread reactivity for synaptophysin (cytoplasmic). You consider the differential of adult type glioblastoma, *IDH* wild type versus diffuse pediatric type glioma, H3 wild type / *IDH* wild type. Which of the following is the best next step for a definitive diagnosis?

1. Send for global methylation profiling
2. Send for *MGMT* promoter methylation profiling
3. Send for next generation DNA sequencing
4. Send for NGS RNA sequencing (RNAseq)

Board review style answer #1

**A.** Send for global methylation profiling. There is significant overlap in the histological and genetic molecular features between adult type glioblastoma, *IDH* wild type and pediatric type high grade glioma H3 wild type / *IDH* wild type. Global methylation profiling is the only way to definitively differentiate these entities. Answer B is incorrect because *MGMT* methylation profiling does not help with the diagnosis, although it might be needed for treatment purposes. Board review style question #2

Which of the following is a syndrome associated with pediatric type diffuse high grade glioma H3 wild type / *IDH* wild type?

1. Cowden syndrome
2. Lynch syndrome
3. Neurofibromatosis type 1
4. Von Hippel-Lindau syndrome

Board review style answer #2

**B.** Lynch syndrome. There is an association between pediatric type high grade glioma H3 wild type / *IDH* wild type and Lynch syndrome (DNA mismatch repair gene defects). Another familial syndrome associated with this tumor is Li-Fraumeni syndrome (germline *TP53* mutation), which is not among the answer choices here. Answer A is incorrect because Cowden syndrome (germline *PTEN* mutation) is associated with Lhermitte-Duclos disease and meningioma. Answer D is incorrect because von Hippel-Lindau syndrome (germline *VHL* mutation) is associated with hemangioblastoma within the CNS. Neither syndrome is associated with pHGG, H3 wild type / *IDH* wild type. Answer C is incorrect because neurofibromatosis type 1 is associated with neurofibroma, pilocytic astrocytoma and adult type glioblastoma but not pediatric high grade gliomas.
